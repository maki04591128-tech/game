# 演出設計書

> **📌 実装状況**
>
> 現在はテキスト演出（メッセージログ）と基本的な色分け表示のみ実装済み。
> アニメーション・画面効果は Phase 6 以降で段階的に追加予定。

## 1. 演出方針

| 項目 | 方針 |
|------|------|
| 基本スタイル | テキストベースのローグライクらしさを維持 |
| 視覚演出 | 控えめだが雰囲気のある色彩設計 |
| テキスト演出 | メッセージログによる状況伝達が主軸 |
| アニメーション | 最低限のフィードバック（将来追加） |
| ダークファンタジー感 | 暗色基調、不穏さを感じるUI |

## 2. 実装済み演出

### 2.1 色彩演出

#### ウィンドウ配色（ダークファンタジーテーマ）

| 要素 | 色コード | 印象 |
|------|---------|------|
| ウィンドウ背景 | #1a1a2e | 深い夜の闇 |
| ステータスバー背景 | #16213e | 暗い藍色 |
| マップエリア背景 | #0f0f23 | 深淵の黒 |
| 操作説明バー背景 | #0f3460 | 暗い青 |

#### ステータスバー色分け

| パラメータ | 通常色 | 意味 |
|-----------|--------|------|
| HP | #4ecca3（緑） | 生命力 |
| MP | #00d9ff（水色） | 魔力 |
| SP | #ffd93d（黄） | 活力 |
| 満腹度 | #ff6b6b（赤） | 飢餓の危険度 |
| 正気度 | #c0a0ff（紫） | 精神の不安定さ |
| 階層 | #e94560（赤） | 危険度 |
| レベル/EXP | #ffa500（オレンジ） | 成長 |
| 日時 | #c0a0ff（紫） | 時間の流れ |

#### HP色段階変化

| HP割合 | 色 | 演出意図 |
|--------|-----|---------|
| 76〜100% | 緑 | 安全 |
| 51〜75% | 黄色 | 注意 |
| 26〜50% | オレンジ | 警告 |
| 1〜25% | 赤 | 危険 |

#### 満腹度・正気度色段階変化

段階に応じて表示色が変化し、プレイヤーに視覚的な警告を提供。

### 2.2 マップ演出

| 演出 | 説明 |
|------|------|
| FOV（視界）| 可視→明るい色、探索済み→暗い色、未探索→黒 |

## 3. 将来実装予定の演出

### 3.1 画面効果（Phase 6）

| 演出 | トリガー | 説明 |
|------|---------|------|
| ダメージフラッシュ | 被ダメージ時 | 画面全体が一瞬赤く点滅 |
| クリティカルシェイク | クリティカルヒット | 画面が微振動 |
| レベルアップエフェクト | レベルアップ時 | プレイヤー位置にパーティクル風の光 |
| 死に戻りトランジション | HP0時 | 画面が暗転→白いテキスト→再開 |
| 階段遷移 | 階段使用時 | フェードアウト→フェードイン |

### 3.2 テキスト演出の拡張

| 演出 | 説明 |
|------|------|
| 色分けメッセージ | イベント種別に応じた色（戦闘=赤、回復=緑、システム=黄等） |
| フロートテキスト | ダメージ数値がエンティティ上に一瞬浮かぶ |
| タイプライター効果 | 重要メッセージが1文字ずつ表示される |

### 3.3 アニメーション

| 演出 | 説明 |
|------|------|
| 攻撃モーション | 攻撃方向へのスプライト微移動 |
| 移動アニメーション | タイル間の滑らかな遷移（オプション） |
| 状態異常表示 | キャラクター上に状態アイコンの点滅 |

### 3.4 正気度演出（Phase 5）

| 正気度段階 | 演出 |
|-----------|------|
| Normal（80~100） | 通常表示 |
| Uneasy（60~79） | 画面端が微かにぼやける |
| Anxious（40~59） | メッセージに意味不明な文字が混ざる |
| Unstable（20~39） | マップの色彩が歪む |
| Madness（1~19） | 幻覚（存在しない敵が見える） |
| Broken（0） | 全画面が歪み、操作がランダムに入れ替わる |
