# コミュニティテストチェックリスト

## テスター向けガイド

### テスト環境
- OS: Windows 10/11 (64-bit)
- .NET: 10.0 Runtime
- メモリ: 4GB以上推奨

---

## 基本機能テスト

### 1. キャラクター作成
- [ ] 全10種族（Human/Elf/Dwarf/Orc/Beastfolk/Halfling/Undead/Demon/FallenAngel/Slime）で作成可能
- [ ] 全10職業（Fighter/Knight/Thief/Ranger/Mage/Cleric/Monk/Bard/Alchemist/Necromancer）で作成可能
- [ ] 種族×職業の全組み合わせ（100通り）が正常に動作
- [ ] ステータス初期値が種族・職業に応じて適切

### 2. ダンジョン探索
- [ ] マップ生成が正常（部屋・通路が繋がっている）
- [ ] 階段の昇降が正常に動作
- [ ] ミニマップが正しく表示される
- [ ] 視界（FOV）が正常に機能する
- [ ] 各階層のレベルデザインが適切

### 3. 戦闘システム
- [ ] 物理攻撃が正常に動作
- [ ] 魔法攻撃が正常に動作（6属性: Fire/Ice/Lightning/Poison/Holy/Dark）
- [ ] クリティカルヒットが発生する
- [ ] 状態異常が正常に適用・解除される
- [ ] 敵AIが適切に行動する
- [ ] ボス戦が正常に進行する

### 4. アイテム・装備
- [ ] アイテムの拾得・使用が正常
- [ ] 装備の着脱が正常
- [ ] エンチャント効果が反映される（CriticalBoost/SpeedBoost/DefenseBoost/Thorns等）
- [ ] ポーション全種類が正常に動作
- [ ] クラフティングが正常に動作

### 5. サバイバルシステム
- [ ] 満腹度が正常に変動する（10段階）
- [ ] 渇き度が正常に変動する（10段階）
- [ ] 疲労度が正常に機能する
- [ ] 正気度が正常に変動する
- [ ] 各状態のペナルティが適切

### 6. シンボルマップ・世界探索
- [ ] シンボルマップが正常に表示される
- [ ] 12領地が全て存在する
- [ ] 集落・町・首都が訪問可能
- [ ] ダンジョン入口（盗賊の巣/ゴブリンの巣等）が機能する
- [ ] NPC対話が正常に動作する

### 7. クエストシステム
- [ ] メインクエストが進行可能
- [ ] サブクエストが受注・完了可能
- [ ] クエスト報酬が正常に付与される

### 8. セーブ・ロード
- [ ] 手動セーブが正常に動作
- [ ] オートセーブが正常に動作
- [ ] ロードが正常に動作（データ破損なし）
- [ ] セーブデータのバックアップが作成される
- [ ] 破損データからの復旧が機能する

### 9. 設定・アクセシビリティ
- [ ] 全5難易度が選択可能
- [ ] 色覚モード（5種類）が切り替え可能
- [ ] ハイコントラストモードが機能する
- [ ] ゲーム速度調整が機能する
- [ ] キーバインドが正常に動作する

### 10. チュートリアル・ヘルプ
- [ ] チュートリアル18種が適切なタイミングで表示される
- [ ] ヘルプ26トピックが参照可能
- [ ] ゲームオーバー画面に統計情報が表示される

## バランステスト

### 難易度別プレイスルー
- [ ] Easy: 初心者でもクリア可能なバランスか
- [ ] Normal: 適度な挑戦を提供しているか
- [ ] Hard: 経験者にとって歯応えがあるか
- [ ] Nightmare: 極めて困難だが理不尽ではないか
- [ ] Ironman: 永久死亡の緊張感が適切か

### 成長バランス
- [ ] レベルアップ速度は適切か
- [ ] スキルツリー（160ノード）のバランスは適切か
- [ ] 装備の強さの推移は適切か

---

## パフォーマンステスト

- [ ] ダンジョン生成に5秒以上かからない
- [ ] セーブ処理に2秒以上かからない
- [ ] 大量の敵が表示されても動作が重くならない
- [ ] 長時間プレイでメモリリークが発生しない
