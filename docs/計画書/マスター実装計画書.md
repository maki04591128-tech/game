# ゲーム実装計画書（マスター）

本書はプロジェクト全体の概要・バージョンロードマップを示すマスター計画書です。
各バージョンの詳細タスクは個別の計画書を参照してください。

## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| プロジェクト名 | ろーぐらいくげーむ |
| ジャンル | ターン制ローグライクRPG |
| 開発環境 | C# / .NET 10.0 / WPF / Visual Studio 2022 |

| テストフレームワーク | xUnit 2.7.0 + FlaUI.UIA3 4.0.0 |

## 2. プロジェクト構成

```
RougelikeGame.sln
├── src/
│   ├── RougelikeGame.Core/        # ドメイン層（エンティティ、値オブジェクト、アイテム、マップ、AI）
│   ├── RougelikeGame.Engine/      # エンジン層（戦闘、ダメージ計算、属性、状態異常、ターン管理）
│   ├── RougelikeGame.Data/        # データ層（魔法言語データベース、呪文解析）
│   ├── RougelikeGame.Gui/         # WPF GUI（GameController、描画、ウィンドウ群）
│   └── RougelikeGame.App/         # コンソールアプリ（InteractiveGameDemo）
└── tests/
    ├── RougelikeGame.Core.Tests/  # Coreユニットテスト
    └── RougelikeGame.Gui.Tests/   # Guiユニットテスト
```
