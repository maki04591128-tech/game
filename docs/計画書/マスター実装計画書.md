## 2. プロジェクト構成

```
RougelikeGame.sln
├── src/
│   ├── RougelikeGame.Core/        # ドメイン層（エンティティ、値オブジェクト、アイテム、マップ、AI）
│   ├── RougelikeGame.Engine/      # エンジン層（戦闘、ダメージ計算、属性、状態異常、ターン管理）
│   ├── RougelikeGame.Data/        # データ層（魔法言語データベース、呪文解析）
│   ├── RougelikeGame.Gui/         # WPF GUI（GameController、描画、ウィンドウ群）
│   └── RougelikeGame.App/         # コンソールアプリ（InteractiveGameDemo）
└── tests/
    ├── RougelikeGame.Core.Tests/  # Coreユニットテスト
    └── RougelikeGame.Gui.Tests/   # Guiユニットテスト
```

### 3.2 マイルストーン詳細

#### Ver.prt.0.1（プロトタイプ完成）
- Phase 1〜3 完了 ✅

- Phase 6.1〜6.4（タイトル画面、設定画面、BGM/SE、パフォーマンス最適化）
- **達成条件**: 全ゲームシステムの実装完了

#### Ver.prt.0.2（基盤拡張）
- Ver.prt.0.1 完了後のプレイテストに基づき策定
- 他のシステムの前提となる基盤システムを優先的に実装（14タスク完了）
- 設計書: [21_拡張システム設計書](../企画設計書/21_拡張システム設計書.md)
- **達成条件**: 全14タスク実装完了 ✅

#### Ver.prt.0.3（システム深化）
- Ver.prt.0.2 の基盤の上に成長・戦闘・経済の深みを追加（27タスク完了）
- **達成条件**: 全27タスク実装完了 ✅

#### Ver.prt.0.4（大型新機能）
- パーティ/仲間/拠点/エンドコンテンツ等の大型システム（24タスク完了）
- **達成条件**: 全24タスク実装完了 ✅

#### Ver.prt.0.5（ゲーム完走フロー確立）
- ゲーム開始→ダンジョン探索→ボス戦→クリア→NG+の一連フローを完走可能にする
- ボス敵の定義・配置（6体のフロアボス）
- ゲームクリア判定・クリア画面・スコア計算
- ゲームオーバーメニュー完成
- NG+開始処理
- 主要システムのGameController統合（仲間AI/料理/スキル合成/転職/拠点）
- クエスト自動進行・メインクエストライン
- **達成条件**: ゲームを最初から最後まで一通り遊べる状態（= Ver.prt.1.0）

#### Ver.α.0.1〜2（世界観補強・没入感向上）
- メインストーリー・エンディング分岐テキスト
- NPC人物設定・会話テキスト執筆
- クエスト詳細内容（依頼文、背景ストーリー等）
- 宗教の教義・経典テキスト
- 魔法言語のフレーバーテキスト・碑文内容
- 領地ロケーション描写・アイテム説明文・敵図鑑テキスト

#### Ver.β.0.1〜2（リソース強化）
- ピクセルアートグラフィック（アスキーアートから置換）
- BGM/SE本格実装（suno生成BGM、効果音素材）
- 画面演出・エフェクト（ダメージ表示、属性エフェクト、画面揺れ等）
- UI/UXの洗練（フォント、レイアウト、アニメーション）

#### Ver.0.1〜0.9（テストローンチ）
- Steam対応（Steamworks統合）
- 総合バランス調整
- デバッグ・不具合修正
- 遊びやすさ改善（チュートリアル、ヒント表示、難易度選択）

#### Ver.1.0（正式リリース）
- Steam正式リリース
- 全コンテンツ最終確認

## 4. 開発フェーズ概要

```
[Phase 1: 基盤構築] → [Phase 2: コアシステム] → [Phase 3: GUI・操作] → [Phase 4: ゲームプレイ拡張] → [Phase 5: コンテンツ・バランス] → [Phase 6: リリース準備] → [Ver.prt.0.2-0.4: システム拡張] → [Ver.prt.0.5: ゲーム完走フロー]
```

| フェーズ | 目標 | 状態 | バージョン目標 |
|----------|------|------|---------------|

| Phase 5: コンテンツ・バランス | 種族/職業、宗教、魔法言語、領地、敵バリエーション | ✅ 完了 | Ver.prt.0.1 |
| Phase 6: リリース準備 | タイトル画面、設定、BGM/SE、最適化、Steam | ✅ 完了 | Ver.prt.0.1 |
| Ver.prt.0.2-0.4: システム拡張 | 新システム65タスク（基盤拡張/深化/大型新機能） | ✅ 完了 | Ver.prt.0.2〜0.4 |
