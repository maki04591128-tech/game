# リリースノート Ver.1.0（正式リリース）

## 🎮 ゲーム概要

**ろーぐらいくげーむ** — ターン制ローグライクRPG

## 📋 主要機能

### コアシステム
- ターン制ローグライク戦闘（物理/魔法/状態異常/属性6種）
- BSPダンジョン自動生成（7種類の部屋タイプ、罠/宝箱/隠し通路）
- シンボルマップ（12領地、可変サイズ最大500,000マス）
- 5種族 × 8職業のキャラクター作成
- 63ノードスキルツリー
- 魔法言語（ルーン語）呪文合成システム
- 死に戻り・正気度システム

### ゲームプレイ
- 満腹度/渇き度10段階管理
- 疲労度システム
- クラフティング・料理・鍛冶
- NPC対話・クエストシステム（22クエスト）
- ショップ・宿屋・ギルド・神殿
- ペット・仲間・パーティシステム
- 投資・経済システム

### 世界観
- 12領地（首都/森林/山岳/沿岸/南部/辺境/砂漠/沼地/凍土/湖畔/火山/聖域）
- メインストーリー＋サブクエスト
- NPC80名以上
- 敵種35体以上、アイテム37種以上

### テクニカル
- WPF GUI（GrimDawn風スキルツリー、PoE風装備画面）
- オートセーブ・手動セーブ・バックアップ
- クラウドセーブ対応
- 5段階難易度（Easy/Normal/Hard/Nightmare/Ironman）
- アクセシビリティ（色覚5モード/ハイコントラスト/ゲーム速度調整/スクリーンリーダー）
- チュートリアル18種/ヘルプ26トピック
