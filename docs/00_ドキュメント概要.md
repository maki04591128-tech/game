# ドキュメント概要（新版）

> **📌 本書について**
>
> 本書は新企画書 `docs/01_新企画書.md`（**málfræðingur — Rúnagaldramaðurinn** / Malfraedingur: The Rune Mage）に基づく新ドキュメント体系の目次です。

> 旧ドキュメント体系（137 システム構成）は `docs/archive/` を参照してください。

> **⚠️ 重要な方針（2026-04-17 改訂）**：
> 本プロジェクトは **「コードを削除せず、旧仕様を保全したまま新仕様を追加／拡張する非破壊アプローチ」** を採用します。既存コード・テスト・XAML は残したまま、`GameController` 等からの参照解除とフィーチャフラグで新仕様モードを実現します。詳細は `docs/流用資産マップ.md` を参照。

## 5. コーディング・テストポリシー（凍結部分）

Phase A 時点で維持されるプロジェクト運用規約：

- ソリューション：`RougelikeGame.sln`（プロジェクト名も互換性維持のため継続）
- .NET 10.0 / WPF / xUnit 2.7.0 / FlaUI.UIA3 4.0.0
- テスト実行：
  - 全件：`dotnet test RougelikeGame.sln`
  - GUI オート除外：`--filter "FullyQualifiedName!~GuiAutomationTests&FullyQualifiedName!~GuiSystemVerificationTests"`
- ターミナル UTF-8：PowerShell では `[Console]::OutputEncoding = [System.Text.Encoding]::UTF8; chcp 65001` を先頭に付ける
