## 0. 目的

本作の核となる魔法言語システム（古ノルド語 50 語、7 カテゴリ、文法準拠構文）を定義する。新企画書 §2, §5.1, 付録 B を具体化する。

## 1. 新企画書との対応

- 主な参照先：`docs/01_新企画書.md`

## 2. 流用・改修・新規の方針

- **流用**: `docs/archive/企画設計書_旧/08_魔法言語設計書.md`（79 語設計、構文、カテゴリ定義）を下敷きとして Ver.1.0 向けに 50 語へ絞り込む。
- **改修**: 50 語選定・古ノルド語文法チェック・文法ボーナス／暴発リスクのルールを明文化。
- **新規**: 条件語によるスクリプト詠唱、ネイティブ音声朗読仕様。

## 3. 目次（後続フェーズで肉付け）

1. 用語定義（語／カテゴリ／構文／詠唱）
2. 7 カテゴリ（効果語／対象語／属性語／修飾語／範囲語／時間語／条件語）
3. Ver.1.0 で採用する 50 語リスト（語・意味・カテゴリ・基礎 MP・詠唱コスト）
4. 古ノルド語文法規則（修飾語 → 効果語 → 対象語、活用規則）
5. 文法的正しさによる威力ボーナス／暴発リスク
6. 既存コード（`RougelikeGame.Data/MagicLanguage/*`、`SpellParser`、`SpellCastingSystem`）との対応
7. Ver.1.1〜1.5 での段階拡張（60 → 70 → 79 語）
