## 1. システム概要

### 1.1 コンセプト

本作における「死」は終わりではなく、知識を蓄積しながら攻略を進めるための手段である。
ただし、死に戻りには「正気度」という代償が存在し、無制限にやり直すことはできない。

```
┌─────────────────────────────────────────────────────────────┐
│                    死に戻りシステム概念図                     │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   【1周目】          【2周目】          【3周目】             │
│   正気度: 100        正気度: 85         正気度: 65            │
│      │                 │                  │                 │
│   探索 → 死亡       探索 → 死亡       探索 → ???           │
│      │                 │                  │                 │
│   知識獲得           知識獲得           知識獲得             │
│   ・NPC情報          ・ダンジョン構造    ・ボス弱点           │
│   ・スキル習得       ・魔法言語          ・隠しルート         │
│      │                 │                  │                 │
│      └─────→ 周回 ←─────┘                  │                 │
│                │                          │                 │
│                └──────────────────────────┘                 │
│                                                             │
│   正気度低下 → バッドステータス発生 → 攻略困難化             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 基本ルール

| 項目 | 内容 |
|------|------|
| 死に戻り発動条件 | HP0、餓死、ゲームオーバーイベント、時間切れ |
| 戻る地点 | 初期開始地点（素性により異なる） |
| 正気度減少 | 死に戻り1回につき5~25減少（死因により変動） |
| 引き継ぎ要素 | 知識系（後述）※NPC関係は引き継がれない |
| リセット要素 | 肉体系（後述）+ 世界状態全般 |
| 時間巻き戻し | 同一キャラクターが開始地点に戻る。痕跡・死体・NPC関係は全て消失 |

## 2. 正気度システム

### 2.1 正気度パラメータ

```
正気度 = 0 ~ 100（初期値: 100）

※正気度は死に戻りでのみ減少（通常プレイでは減少しない）
※一部の特殊手段でのみ回復可能
```

### 2.2 正気度の減少

#### 死因別減少量

| 死因 | 減少量 | 備考 |
|------|--------|------|
| 通常戦闘死（Combat） | -5 | 最も軽い |
| ボス戦死亡（Boss） | -8 | やや重い |
| 落下死（Fall） | -8 | 穴・高所から落下 |
| 毒死（Poison） | -7 | 毒ダメージによる死亡 |
| 餓死（Starvation） | -10 | 準備不足のペナルティ |
| 即死トラップ（Trap） | -10 | |
| 原因不明（Unknown） | -10 | その他の死因 |
| 時間切れ（TimeLimit） | -15 | 大幅減少 |
| 呪い・特殊死（Curse） | -15 | 呪いによる死亡 |
| 自殺（Suicide） | -20 | 非常に重い |
| 正気度関連死（SanityDeath） | -25 | 狂気状態での死（最も重い） |

#### 減少量の補正

| 補正要素 | 効果 |
|----------|------|
| アンデッド種族 | 減少量-20% |
| 死神信仰（高位） | 減少量-30% |
| 特定スキル「鋼の精神」 | 減少量-10% |
| 特定アイテム「魂の護符」 | 減少量-5（1回限り） |
| 堕天使種族 | 減少量+50% |
| 呪いアイテム装備中 | 減少量+20% |

### 2.3 正気度の回復

正気度の回復は非常に困難であり、限られた手段でのみ可能。

| 回復方法 | 回復量 | 条件・制約 |
|----------|--------|------------|
| 世界樹の雫 | +10 | 超希少アイテム、1周回で1個のみ |
| 光の神殿の奇跡 | +5 | 最高位の信仰度が必要 |
| 賢者院の儀式 | +5 | 大量の金と素材が必要 |
| 特定素性クリア | +15 | 「贖罪者」素性の完遂 |
| 隠しエンディング | +全回復 | 真のエンディング到達 |

## 3. 正気度段階と影響

### 3.1 正気度段階

| 段階 | 正気度範囲 | 状態名 | 概要 |
|------|------------|--------|------|
| 5 | 80~100 | 正常（Normal） | 通常状態、ペナルティなし |
| 4 | 60~79 | 動揺（Uneasy） | 軽微な影響が出始める |
| 3 | 40~59 | 不安（Anxious） | 明確なデバフが発生 |
| 2 | 20~39 | 不安定（Unstable） | 深刻なデバフ、幻覚発生 |
| 1 | 1~19 | 狂気（Madness） | ゲームプレイに重大な支障 |
| 0 | 0 | 廃人化（Broken） | 実質ゲームオーバー（救済あり） |

### 3.2 段階別詳細効果

#### 段階5: 正常（80~100）
| 効果 | 内容 |
|------|------|
| ペナルティ | なし |
| 特記事項 | 全ての機能が正常に動作 |

---

#### 段階4: 動揺（60~79）
| 効果 | 内容 |
|------|------|
| 会話変化 | 一部NPCが主人公の異変に気づく |
| 睡眠障害 | 休息による回復効率-10% |
| 悪夢 | 休息時に低確率でイベント発生 |
| メタ演出 | UIに微細なノイズが入ることがある |

---

#### 段階3: 不安（40~59）
| 効果 | 内容 |
|------|------|
| 精神耐性低下 | 精神系状態異常耐性-20% |
| 幻聴 | 時々存在しない音が聞こえる（テキスト表示） |
| 判断力低下 | 一部選択肢が見えなくなる |
| NPC反応変化 | 一部NPCが距離を置く、価格上昇 |
| 被害妄想 | 中立NPCが敵に見えることがある（誤認） |
| メタ演出 | 画面端に影が映ることがある |

---

#### 段階2: 不安定（20~39）
| 効果 | 内容 |
|------|------|
| 幻覚 | 存在しない敵が出現（攻撃しても消えない、ダメージなし） |
| 幻聴 | 頻繁にノイズメッセージが表示される |
| 記憶混濁 | マップの一部が表示されなくなる |
| 暴走 | 20%の確率で行動がランダムに |
| パラノイア | 味方NPCを敵と誤認する可能性 |
| 自傷衝動 | 戦闘中に10%で自分を攻撃 |
| NPC拒絶 | 多くのNPCが会話を拒否 |
| メタ演出 | 画面が歪む、テキストが化ける |

**特殊イベント**: 「内なる声」が話しかけてくる（重要な伏線）

---

#### 段階1: 狂気（1~19）
| 効果 | 内容 |
|------|------|
| 常時幻覚 | 常に幻の敵が複数出現 |
| 操作困難 | 入力が30%の確率で無視される |
| 敵味方混同 | NPCと敵の区別がつかない |
| 自傷常態化 | 毎ターン5%で自傷 |
| 施設利用不可 | ほとんどの施設が利用できない |
| 会話不能 | 会話が支離滅裂になる |
| メタ演出 | 画面全体が不安定、BGMが歪む |

**警告**: この状態での死亡は正気度を大幅に減少させる

---

#### 段階0: 廃人化（0）
| 効果 | 内容 |
|------|------|
| 操作不能 | プレイヤーの入力を完全に無視 |
| 自動行動 | キャラクターが勝手に徘徊 |
| 終焉 | 一定時間後に衰弱死 |
| **知識消失** | スキル・魔法言語・宗教・各種知識が全て消失 |
| **フラグ保持** | 重要フラグと実績のみ保持される |

**救済措置**:
- 「死に戻りの案内人」による強制介入イベント
- 正気度を20まで回復
- **ただし強化要素（スキル・魔法言語・宗教等）は全て消失**
- フラグのみが残り、一部のショートカットが可能
- 救済は1セーブデータで3回まで
- 3回使い切り後の廃人化は**真のゲームオーバー（セーブデータ削除）**

## 4. 引き継ぎシステム

### 4.1 引き継ぎ要素（知識）

死に戻り後も維持される要素。これらが攻略の鍵となる。

| カテゴリ | 引き継ぎ内容 | 詳細 | 正気度0時 |
|----------|--------------|------|-----------|
| スキル | 習得したスキル | 全て引き継ぎ | **消失** |
| 魔法言語 | 習得した語彙 | 全て引き継ぎ | **消失** |
| 宗教 | 入信状態、信仰度 | 信仰度は80%引き継ぎ | **消失** |
| NPC関係 | NPCとの関係・記憶 | **引き継がれない（時間巻き戻しのため）** | **消失** |
| マップ知識 | 探索したマップ情報 | 構造は変化するが傾向は把握 | **消失** |
| イベント知識 | 体験したイベント記録 | 選択肢の結果を記憶 | **消失** |
| レシピ | 習得した調合・鍛冶レシピ | 全て引き継ぎ | **消失** |
| フラグ | 特定のストーリーフラグ | 重要フラグのみ | **保持** |
| 実績 | 達成した実績 | 永続的に引き継ぎ | **保持** |

### 4.2 正気度0時のペナルティ（廃人化）

正気度が0になった場合、通常の引き継ぎ要素の大部分を失う重大なペナルティが発生する。

```
┌─────────────────────────────────────────────────────────────┐
│                   正気度0時（廃人化）                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【消失する要素】                 【保持される要素】          │
│  ・スキル（全て）                 ・フラグ（重要フラグのみ）   │
│  ・魔法言語（全て）               ・実績                      │
│  ・宗教（入信状態・信仰度）                                   │
│  ・NPC知識                                                   │
│  ・マップ知識                                                │
│  ・イベント知識                                              │
│  ・レシピ                                                    │
│                                                             │
│  → 実質的にほぼ最初からやり直し                              │
│  → ただしフラグにより一部ショートカット可能                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 保持されるフラグの例
| フラグ | 効果 |
|--------|------|
| 隠しロケーション発見済み | 発見場所のヒントが出る |
| ボス撃破済み | 弱点の断片的な記憶が残る |
| 真実の一端を知った | 特定の選択肢が最初から出現 |
| NPCの本性を見た | 警戒すべきNPCに違和感を覚える |

#### 救済措置
- 廃人化からの復帰時、正気度は20まで回復
- 救済は1セーブデータで3回まで
- 3回の救済を使い切ると**真のゲームオーバー（セーブデータ削除）**

### 4.3 リセット要素（肉体）

死に戻り後にリセットされる要素。

| カテゴリ | リセット内容 | 詳細 |
|----------|--------------|------|
| レベル | 1に戻る | 経験値も0 |
| ステータス | 初期値に戻る | 種族・クラス基準 |
| HP/MP/SP | 最大値・現在値とも初期化 | |
| 装備 | 素性の初期装備に戻る | |
| アイテム | 素性の初期アイテムに戻る | |
| 所持金 | 素性の初期所持金に戻る | |
| NPC関係 | 全リセット | 記憶・噂・好感度全て消失（時間巻き戻し） |
| クエスト進行 | リセット | フラグ依存で変化 |
| ダンジョン構造 | 再生成 | シード値変更 |
| 時間 | 1日目に戻る | ターン制限リセット（1年 = 31,536,000ターン） |
| 痕跡・死体 | 全消失 | 自身の痕跡は残らない |

### 4.4 特殊引き継ぎ

| 要素 | 条件 | 効果 |
|------|------|------|
| 魂の装備 | 特殊エンチャント付き装備 | 装備ごと引き継ぎ |
| 記憶の結晶 | 使用すると1アイテム保持 | 消費アイテム |
| 輪廻の指輪 | 装備中は所持金10%保持 | 希少装備 |
| 約束の証 | NPCとの特別な約束 | 好感度を一部保持 |

## 6. ゲームプレイへの影響

### 6.1 正気度による難易度変化

| 正気度 | 実質難易度 | プレイスタイル |
|--------|------------|----------------|
| 80~100 | Normal | 通常プレイ |
| 60~79 | Normal+ | わずかに難化 |
| 40~59 | Hard | 戦略的プレイが必要 |
| 20~39 | Very Hard | 高い技量が必要 |
| 1~19 | Nightmare | クリア困難 |

### 6.2 低正気度プレイの戦略

低正気度でも攻略可能にするための要素：

| 要素 | 内容 |
|------|------|
| 幻覚の見分け方 | ダメージを与えても減らない＝幻覚 |
| 自傷対策 | 防具で軽減、特定スキルで無効化 |
| NPC協力 | 一部NPCは狂気状態でも協力 |
| 隠しルート | 狂気状態専用の選択肢・ルート |
| 狂気の力 | 正気度20以下で解放される特殊スキル |

### 6.3 周回プレイの推奨サイクル

```
【効率的な周回の例】

1周目（正気度100→85）
  目標: 世界の把握、基本スキル習得
  推奨: 安全な場所の探索、NPC情報収集

2周目（正気度85→70）
  目標: 主要ダンジョンの攻略開始
```
