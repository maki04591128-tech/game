### 1.2 テスト方針

| 方針 | 説明 |
|------|------|
| 全テスト必須 | コード変更のたびに全テストを実行 |
| 乱数固定 | `TestRandomProvider(fixedValue)` で再現可能なテスト |
| GUIテスト除外 | 通常テスト実行時は `--filter "FullyQualifiedName!~GuiAutomationTests"` |
| 日本語対応 | テスト実行前に UTF-8 エンコーディングを設定 |

private readonly TestRandomProvider _fixedRandom = new(0.5);

```csharp
// GameControllerを初期化して統合的にテスト
private GameController CreateInitializedController()

    var controller = new GameController();
    controller.Initialize();
    return controller;

```

### 5.3 FlaUI GUIオートメーションテスト

```csharp
// FlaUI でアプリを起動し、UI Automation で要素を取得・検証
using var app = Application.Launch(AppPath);
using var automation = new UIA3Automation();
var window = app.GetMainWindow(automation, TimeSpan.FromSeconds(10));
// TextBlock等の要素を AutomationId や Name で検索
var statusBar = window.FindFirstDescendant(cf => cf.ByAutomationId("StatusBarText"));
```

| 空腹度テスト | HungerDecayInterval=600ターン。大きいマップではレンダリング負荷で待ち時間のマージンを広めに取る必要あり |
