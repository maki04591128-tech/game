# キャラクター設計書

> **📌 実装状況**
>
> 種族・職業・素性の**定義データとPlayer統合**は Phase 4（タスク 4.15）で実装済みです。
> Phase 5 で種族特性ゲーム効果接続、成長曲線、スキルシステム、クラフトシステム、チュートリアルシステムを実装済み。
>
> | 項目 | 実装状況 |
> |------|----------|
> | 種族列挙型（10種） | ✅ `Race` 列挙型（Human/Elf/Dwarf/Orc/Beastfolk/Halfling/Undead/Demon/FallenAngel/Slime） |
> | 職業列挙型（10種） | ✅ `CharacterClass` 列挙型（Fighter/Knight/Thief/Ranger/Mage/Cleric/Monk/Bard/Alchemist/Necromancer） |
> | 素性列挙型（10種） | ✅ `Background` 列挙型（Adventurer/Soldier/Scholar/Merchant/Peasant/Noble/Wanderer/Criminal/Priest/Penitent） |
> | 種族定義（RaceDefinition） | ✅ StatBonus/HpBonus/MpBonus/Traits/ExpMultiplier/SanityLossMultiplier 定義済み |
> | 職業定義（ClassDefinition） | ✅ StatBonus/HpBonus/MpBonus/InitialSkills 定義済み |
> | 素性定義（BackgroundDefinition） | ✅ StartingGold/StatBonus 定義済み |
> | Player.Race/CharacterClass/Background | ✅ プロパティ追加済み。`Player.Create(name, race, class, background)` オーバーロードで初期化 |
> | ステータスボーナス適用 | ✅ 種族/職業/素性の StatModifier が Player 作成時に BaseStats に適用される |
> | SaveData対応 | ✅ `PlayerSaveData` に Race/CharacterClass/Background フィールド追加済み |
> | 種族特性のゲーム効果接続 | ✅ RacialTraitSystemをPlayer/CombatSystem/GameControllerに統合済み (Phase 5) |
> | 種族/職業ごとの成長曲線 | ✅ GrowthSystemによる種族/職業別成長曲線実装済み (Phase 5) |
> | スキルシステム | ✅ SkillSystemをGameControllerに統合、UseSkillアクション実装済み (Phase 5) |
> | 素性クリア条件システム | ✅ BackgroundClearSystem作成、フラグ追跡・クリア条件判定 (Phase 5) |
> | クラフトシステム | ✅ CraftingSystem（10レシピ、強化+10、エンチャント）(Phase 5) |
> | チュートリアルシステム | ✅ TutorialSystem（18ステップ/18トリガー）(Phase 5) |
> | キャラクター作成GUI画面 | ✅ `CharacterCreationWindow`（10種族/10職業/10素性選択、名前入力、ステータスプレビュー）実装済み |
>
> **Player 初期化（デフォルト）**: `Player.Create("name")` → Human/Fighter/Adventurer、`BaseStats = Stats.Default`（全10）
> **Player 初期化（指定）**: `Player.Create("name", Race.Elf, CharacterClass.Mage, Background.Scholar)` → 種族/職業/素性の StatBonus 適用
>
> **実装コードの参照先**: `src/RougelikeGame.Core/Systems/CharacterCreation.cs`, `src/RougelikeGame.Core/Entities/Player.cs`, `src/RougelikeGame.Core/Enums/Enums.cs`, `src/RougelikeGame.Core/Systems/CraftingSystem.cs`, `src/RougelikeGame.Core/Systems/SmithingSystem.cs`

### 1.1 作成フロー

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│ 種族選択 │ → │クラス選択│ → │ 素性選択 │ → │ 能力値  │
│         │    │         │    │         │    │  調整   │
└─────────┘    └─────────┘    └─────────┘    └─────────┘
                                                   │
┌─────────┐    ┌─────────┐    ┌─────────┐         │
│ゲーム開始│ ← │ 確認    │ ← │外見・名前│ ←───────┘
│         │    │         │    │  設定   │
└─────────┘    └─────────┘    └─────────┘
```

### 1.2 作成時の決定事項

| 項目 | 決定内容 | 影響 |
|------|----------|------|
| 種族 | 基礎ステータス、種族特性、制約 | 恒久的 |
| クラス | 初期スキル、成長傾向、装備適性 | 恒久的 |
| 素性 | ゲーム目的、開始地点、初期装備、NPC関係 | 恒久的 |

| 宗教 | （ゲーム開始後に選択可能） | 変更可（ペナルティあり） |

## 3. クラス（職業）

### 3.1 クラス一覧

> **⚠️ 設計と実装の差異**: 本設計書の3.2クラス詳細では世界観に基づく設計段階の職業名（狂戦士、召喚士等）を記述していますが、
> 実装では `CharacterClass` 列挙型として Fighter/Knight/Thief/Ranger/Mage/Cleric/Monk/Bard/Alchemist/Necromancer を使用しています。
> 設計段階の狂戦士（Berserker）・召喚士（Summoner）は、実装では騎士（Knight）・吟遊詩人（Bard）に変更されています。

### 4.2 素性別ターン制限

各素性には **1年（31,536,000ターン）** の時間制限があり、素性ごとに制限の背景となる世界的危機が設定されている。
特定フラグの達成により延長・撤廃が可能（達成難易度は周回前提）。

| 素性名 | 危機の種類 | 延長フラグ | 撤廃フラグ |
|--------|-----------|-----------|-----------|
| 冒険者 | ダンジョンのスタンピード | ダンジョン中層ボス撃破（3体中2体） | 全階層ボス撃破＋核の封印 |
| 復讐者 | 仇敵の逃亡計画 | 仇敵の協力者を3人排除 | アジト特定＋逃亡ルート封鎖 |
| 追放貴族 | 追手の包囲網完成 | 3つの領地で協力者を確保 | 全領地の安全拠点確立＋追手の指揮官排除 |
| 逃亡者 | 追手の包囲網完成 | 3つの領地で協力者を確保 | 全領地の安全拠点確立＋追手の指揮官排除 |
| 学者 | 古代遺跡の崩壊 | 古代碑文の解読（6箇所中4箇所） | 全碑文解読＋中枢遺跡への到達 |
| 信仰者 | 裂け目の拡大 | 裂け目の一部封印（6箇所中3箇所） | 全裂け目封印＋異界の門番撃破 |
| 傭兵 | 大戦争の勃発 | 3つの領地で領主の信頼獲得 | 全領地間の和平交渉成立＋調停者の称号獲得 |
| 商人 | 経済崩壊の進行 | 3つの領地で交易路を開通 | 全領地の経済圏統合＋商業ギルド再建完了 |
| 罪人 | 処刑までの猶予終了 | 贖いの試練を3つ達成 | 全試練達成＋裁判官への弁明成功 |
| 運命の子 | 終末の預言成就 | 終末の兆候を3つ阻止 | 全兆候阻止＋終末の根源への到達 |

> **延長効果**: +半年（+15,768,000ターン）、1回のみ
> **撤廃効果**: ターン制限の完全撤廃（無制限プレイ）
> **詳細**: 各素性の危機背景の詳細は `docs/企画設計書/18_設定資料.md` セクション7.4 を参照

## 5. ステータス

### 5.1 基本ステータス

| ステータス | 略称 | 効果 |
|------------|------|------|
| 筋力 | STR | 物理攻撃力、持てる重量 |
| 体力 | VIT | 最大HP、物理防御、毒耐性 |
| 敏捷 | AGI | 回避率、行動速度、移動速度 |
| 器用 | DEX | 命中率、クリティカル率、罠解除 |
| 知力 | INT | 魔法攻撃力、魔法習得速度 |
| 精神 | MND | 最大MP、魔法防御、状態異常耐性 |
| 感覚 | PER | 索敵範囲、隠し発見、先制率 |
| 魅力 | CHA | NPC好感度、価格交渉、一部スキル |
| 幸運 | LUK | ドロップ率、クリティカル、回避 |

### 5.2 派生ステータス

| ステータス | 計算式（仮） |
|------------|--------------|
| 最大HP | 基礎値 + (VIT × 10) + 種族補正 + クラス補正 |
| 最大MP | 基礎値 + (MND × 5) + (INT × 2) + 種族補正 |
| 物理攻撃 | 武器攻撃力 + (STR × 2) |
| 物理防御 | 防具防御力 + (VIT × 1.5) |
| 魔法攻撃 | 杖攻撃力 + (INT × 3) |
| 魔法防御 | 防具魔防 + (MND × 2) |
| 命中率 | 基礎80% + (DEX × 1%) |
| 回避率 | 基礎5% + (AGI × 0.5%) + (LUK × 0.2%) |
| クリティカル率 | 基礎5% + (DEX × 0.3%) + (LUK × 0.5%) |

### 5.3 初期ステータス配分

- 全種族共通基礎値: 各ステータス 10
- 種族補正: 上記種族詳細参照
- ボーナスポイント: 10ポイント（自由に振り分け）
- 1ポイント = 1ステータス上昇
