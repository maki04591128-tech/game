## 1. 戦闘システム概要

### 1.1 基本ルール

| 項目 | 内容 |
|------|------|
| 戦闘形式 | ターン制、1行動=1ターン |
| 戦闘フィールド | プレイマップ上でシームレスに発生 |
| 行動順 | 敏捷値による行動順決定（同値は乱数） |
| 戦闘終了条件 | 敵全滅、逃走成功、HP0（死亡/気絶） |

### 1.2 戦闘の流れ

```
[索敵フェーズ]
    │
    ├─ 敵発見 → 先制判定
    │            ├─ 成功: プレイヤー先攻
    │            └─ 失敗: 敵先攻 or 通常開始
    │
[戦闘フェーズ] ← ループ
    │
    ├─ 行動順決定（敏捷値ベース）
    │
    ├─ 各キャラクターの行動選択
    │   ├─ 移動
    │   ├─ 攻撃（通常/スキル/魔法）
    │   ├─ 防御
    │   ├─ アイテム使用
    │   ├─ 待機
    │   └─ 逃走
    │
    ├─ 行動解決（ダメージ計算、効果適用）
    │
    └─ 戦闘終了判定
         ├─ 敵全滅 → 勝利
         ├─ プレイヤー死亡 → 死に戻り
         └─ 継続 → 次ターンへ
```

## 2. ダメージ計算式

### 2.1 物理ダメージ

```
基礎ダメージ = (攻撃力 × スキル倍率) - (防御力 × 0.5)
最終ダメージ = 基礎ダメージ × 乱数(0.9~1.1) × 属性相性 × クリティカル倍率

※最低ダメージ = 1（0以下にはならない）
```

**詳細パラメータ**:
| 項目 | 計算式 |
|------|--------|
| 攻撃力 | 武器攻撃力 + (STR × 2) + バフ補正 |
| 防御力 | 防具防御力 + (VIT × 1.5) + バフ補正 |
| スキル倍率 | 通常攻撃=1.0、スキルにより1.2~3.0 |
| クリティカル倍率 | 通常=1.0、クリティカル=1.5~2.0 |

### 2.2 魔法ダメージ

```
基礎ダメージ = (魔法攻撃力 × スキル倍率 × 魔法言語補正) - (魔法防御 × 0.3)
最終ダメージ = 基礎ダメージ × 乱数(0.9~1.1) × 属性相性

※魔法は必中（回避不可、ただし魔法耐性で軽減）
```

**詳細パラメータ**:
| 項目 | 計算式 |
|------|--------|
| 魔法攻撃力 | 杖攻撃力 + (INT × 3) + バフ補正 |
| 魔法防御 | 防具魔防 + (MND × 2) + バフ補正 |
| 魔法言語補正 | 語彙の組み合わせにより1.0~2.5 |

### 2.3 属性相性表

| 攻撃属性 | 有利（×1.5） | 不利（×0.5） | 無効 |
|----------|--------------|--------------|------|
| 火 | 氷、植物 | 水、火 | - |
| 水 | 火、雷 | 氷、植物 | - |
| 氷 | 水、風 | 火、氷 | - |
| 雷 | 水、金属 | 地、雷 | - |
| 地 | 雷、火 | 風、植物 | - |
| 風 | 地、植物 | 氷、雷 | - |
| 光 | 闇、アンデッド | 光 | - |
| 闇 | 光、精神 | 闇 | アンデッド |
| 無 | - | - | - |

## 3. 命中・回避・クリティカル

### 3.1 命中判定

```
命中率 = 基礎命中率 + (DEX × 1%) - 敵回避率 + 補正

判定: 乱数(0~100) < 命中率 → 命中
```

| 攻撃タイプ | 基礎命中率 |
|------------|------------|
| 近接攻撃 | 90% |
| 遠距離攻撃 | 80% |
| 魔法攻撃 | 100%（必中） |
| スキル攻撃 | スキルにより変動 |

### 3.2 回避判定

```
回避率 = 基礎回避率 + (AGI × 0.5%) + (LUK × 0.2%) + 装備補正

※回避上限 = 75%（絶対命中25%は保証）
```

| 防具タイプ | 基礎回避率 |
|------------|------------|
| 重装 | 0% |
| 中装 | 5% |
| 軽装 | 10% |
| ローブ | 15% |
| 裸 | 20% |

### 3.3 クリティカル判定

```
クリティカル率 = 基礎5% + (DEX × 0.3%) + (LUK × 0.5%) + 武器補正 + スキル補正

クリティカルダメージ = 通常ダメージ × 1.5
※クリティカル特化ビルドでは最大×2.5まで上昇可能
```

## 4. HP・MP・リソース管理

### 4.1 HP（ヒットポイント）

```
最大HP = 基礎値(50) + (VIT × 10) + (STR × 2)

※ 将来拡張予定: + (レベル × 5) + 種族補正 + クラス補正
```

**HP状態**:
| 状態 | HP割合 | 効果 |
|------|--------|------|
| 健康 | 76~100% | 通常状態 |
| 負傷 | 51~75% | 一部行動にペナルティ |
| 重傷 | 26~50% | 攻撃力-10%、移動速度-20% |
| 瀕死 | 1~25% | 攻撃力-25%、移動速度-50%、一部スキル使用不可 |
| 死亡 | 0% | 行動不能、死に戻り発動 |

### 4.2 MP（マジックポイント）

```
最大MP = 基礎値(20) + (MND × 5) + (INT × 3)

※ 将来拡張予定: + (レベル × 2) + 種族補正
```

**MP回復**:
| 方法 | 回復量 |
|------|--------|
| 自然回復（10ターンごと） | 最大MPの1% |
| 休息（100ターン消費） | 最大MPの10% |
| 瞑想スキル | 最大MPの20% |
| マナポーション | 固定値（アイテムによる） |

### 4.3 スタミナ（SP）

```
最大SP = 100 + (VIT × 2)

※ VIT依存に変更済み（当初設計は固定100）
```

**スタミナ消費**:
| 行動 | 消費SP |
|------|--------|
| 通常移動 | 0 |
| ダッシュ移動 | 2/マス |
| 通常攻撃 | 5 |
| 物理スキル | 10~30 |
| 防御 | 3 |
| 回避行動 | 8 |
| 重装備着用時 | 全消費+20% |

**スタミナ回復**:
| 方法 | 回復量 |
|------|--------|
| 待機（1ターン） | 10 |
| 食事 | 全回復 |
| 自然回復 | 5/10ターン |

### 4.4 満腹度

```
最大満腹度 = 100
```

| 満腹度 | 状態 | HungerStage | 効果 |
|--------|------|-------------|------|
| 80〜100 | 満腹 | Full | スタミナ回復+20% |
| 50〜79 | 普通 | Normal | 通常状態 |
| 25〜49 | 空腹 | Hungry | スタミナ回復-20% |
| 1〜24 | 飢餓 | Starving | HP/SP自然回復停止 |
| 0 | 餓死 | Famished | 死に戻り発動 |

**満腹度消費**:
| 行動 | 消費量 |
|------|--------|
| 自然減少 | 1/600ターン（10分ごと） |
| 激しい戦闘 | 追加で1~3 |
| 休息 | 追加で2 |
| 種族による差異 | ±20~50% |

## 10. Phase 5 追加: AI行動・バランス設定

### 10.1 敵AI行動パターン

| AI行動 | クラス | 特徴 |
|---------|---------|------|
| 基本行動 | BasicMeleeBehavior | 接近→近接攻撃、HP低下で逃走 |
| 遠距離 | RangedBehavior | 射程内で遠距離攻撃、射程外で接近 |
| 狂戦士 | BerserkerBehavior | HP低下で攻撃力上昇、常に接近攻撃、逃走なし |
