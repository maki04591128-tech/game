## 1. システム概要

### 1.1 コンセプト

本作の魔法システムは「古代魔法言語（ルーン語）」を基盤とする。
この言語は古ノルド語をモチーフとしており、九つの世界を繋ぐ力を持つとされる。
プレイヤーは語彙（ルーン語）を習得し、それらを組み合わせることで魔法を構築する。

```
┌─────────────────────────────────────────────────────────────┐
│                    魔法言語システム概念図                     │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   【語彙の習得】                                             │
│   賢者院、ルーン碑文、古代の書 → ルーン語を学ぶ              │
│                                                             │
│   【魔法の構築】                                             │
│   [効果語] + [対象語] + [修飾語] → 詠唱文（ガルドル）         │
│                                                             │
│   例: "brenna" + "fjandi" + "mikill"                         │
│       → 「Mikill brenna fjandi!」 = 強力な炎の魔法           │
│                                                             │
│   【発動】                                                   │
│   詠唱の複雑さ → MP消費・詠唱ターン決定                      │
│   語彙の理解度 → 成功率・威力に影響                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 世界設定：ルーン語の起源

- **九界の言葉**: オーディンが世界樹ユグドラシルで得た知識を元に編み出された
- **ルーン文字との関係**: 刻まれたルーン文字は静的な力を持ち、詠唱（ガルドル）は動的な力を発揮する
- **大崩壊との関係**: かつては全ての語彙が知られていたが、大崩壊で多くが失われた
- **賢者院**: 残された語彙を研究・保存している唯一の機関

### 1.3 基本ルール

| 項目 | 内容 |
|------|------|
| 語彙上限 | なし（習得した分だけ使用可能） |
| 最小構成 | 効果語1つで発動可能 |
| 最大構成 | 7語まで（それ以上は詠唱失敗） |
| 詠唱時間 | 語数×5ターン（基本）、修飾語で変動 |
| MP消費 | 語の種類と数で決定 |
| 失敗リスク | 語の理解度が低いと暴発の可能性 |

| 対象語 | Mál | 効果の対象を指定 | △ | fjandi(敵), sjálfr(自分) |
| 属性語 | Náttúra | 属性を付与・変更 | × | eldr(火), íss(氷) |

### 2.2 構文ルール（ガルドル文法）

古ノルド語の語順に基づき、**修飾語 → 効果語 → 対象語** の順で詠唱する。

```
基本構文: (修飾語) + [効果語] + (対象語) + (追加修飾)

例1: 「Brenna!」（ブレンナ）
     → 最もシンプル、前方1マスに火属性ダメージ
     → 消費: 5ターン

例2: 「Brenna fjandi!」（ブレンナ・フィヤンディ）
     → 「敵を燃やす」、最も近い敵に火属性ダメージ
     → 消費: 10ターン

例3: 「Mikill brenna fjandi!」（ミキル・ブレンナ・フィヤンディ）
     → 「敵を激しく燃やす」、強化版
     → 消費: 15ターン

例4: 「Víðr mikill brenna fjandi!」（ヴィーズル・ミキル・ブレンナ・フィヤンディ）
     → 「周囲の敵を激しく燃やす」、範囲強化
     → 消費: 20ターン

例5: 「Víðr mikill brenna myrk-eldr fjandi langr!」
     → 「周囲の敵を闇炎で激しく持続的に燃やす」
     → 消費: 30ターン
```

## 4. 魔法構築システム

### 4.1 詠唱ターン計算

```
基本詠唱ターン = Σ(各語の詠唱ターン) + Σ(修飾語の詠唱加算)

速度修飾語による補正:
- skjótr(速い): 基本詠唱 × 0.7
- hraðr(急速): 基本詠唱 × 0.5
- þegar(即座): 基本詠唱 = 5ターン（最小固定）

最小詠唱ターン: 5ターン
最大詠唱ターン: 100ターン

例: 「Mikill brenna fjandi」
    = brenna(5) + fjandi追加(0) + mikill(5)
    = 10ターン

例: 「Skjótr mikill brenna fjandi」
    = (5 + 0 + 5) × 0.7
    = 7ターン（切り上げ）

例: 「Þegar brenna」
    = 5ターン（即座修飾で最小化）
```

### 4.2 構築インターフェース

```
┌─────────────────────────────────────────────────────────────┐
│                  ガルドル構築（魔法詠唱）                     │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【習得ルーン語】                                            │
│  ┌─────────┬─────────┬─────────┬─────────┬─────────┐        │
│  │ brenna  │ frysta  │ græða   │ fjandi  │ sjálfr  │ ...   │
│  │ (燃やす) │ (凍らす) │ (癒す)  │ (敵)    │ (自分)  │        │
│  └─────────┴─────────┴─────────┴─────────┴─────────┘        │
│                                                             │
│  【構築中のガルドル】                                        │
│  ┌─────────────────────────────────────┐                    │
│  │ Mikill brenna fjandi!               │                    │
│  │ 「激しく敵を燃やす」                 │                    │
│  └─────────────────────────────────────┘                    │
│                                                             │
│  【予測効果】                                                │
│  ・効果: 火属性ダメージ（威力150%）                          │
│  ・対象: 最も近い敵1体                                       │
│  ・消費MP: 12                                                │
│  ・詠唱: 10ターン（10秒）                                    │
│  ・成功率: 95%                                               │
│                                                             │
│  [登録] [テスト発動] [キャンセル]                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 4.3 MP計算式

```
消費MP = Σ(各語の基礎MP) × Π(乗算係数) × 理解度補正

例: 「Mikill brenna fjandi」
    = (brenna:5 + fjandi:2) × (mikill:1.3) × (理解度補正:1.0)
    = 7 × 1.3 × 1.0
    = 9.1 → 10MP（切り上げ）
```

### 4.4 成功率計算

```
成功率 = 100% - Σ(未熟語彙ペナルティ) - 語数ペナルティ

未熟語彙ペナルティ:
  理解度100%: 0%
  理解度80%: -2%
  理解度60%: -5%
  理解度40%: -10%
  理解度20%: -20%

語数ペナルティ:
  1~3語: 0%
  4語: -3%
  5語: -8%
  6語: -15%
  7語: -25%
```

## 5. 語彙の習得と理解度

### 5.1 習得方法

| 方法 | 内容 | 習得数 |
|------|------|--------|
| 賢者院の講義 | MPを払って学習 | 確実に1語 |
| ルーン碑文の解読 | 古代遺跡で発見 | 1~3語 |
| 古写本の研究 | アイテム消費 | 1~2語 |
| NPCから教わる | クエスト報酬等 | 1語 |
| 実践での閃き | 戦闘中に低確率 | 1語 |
| ヴェルンドの祝福 | 神殿での祈り | 上級語彙 |
| 死に戻り引き継ぎ | 前周回の知識 | 全て |

### 5.2 理解度システム

```
理解度 = 0% ~ 100%

習得時: 20%からスタート
使用ごとに: +1~5%上昇（成功時）
失敗時: +0.5%上昇
最大: 100%
```

**理解度による影響**:
| 理解度 | 威力補正 | 成功率補正 | MP効率 |
|--------|----------|------------|--------|
| 0~20% | ×0.5 | -20% | ×1.5 |
| 21~40% | ×0.7 | -10% | ×1.3 |
| 41~60% | ×0.85 | -5% | ×1.1 |
| 61~80% | ×0.95 | -2% | ×1.0 |
| 81~99% | ×1.0 | 0% | ×1.0 |
| 100% | ×1.1 | +5% | ×0.9 |

## 8. ルーン語と世界設定

### 8.1 設定上の背景

- **九界の言葉**: オーディンが世界樹ユグドラシルに9日間吊るされて得た知識
- **ルーン文字**: フサルク文字に基づく24のルーン文字が基本
- **大崩壊**: ラグナロク級の災厄で多くの高位語彙が失われた
- **賢者院**: 北欧神話の知識を継承し、残った語彙を研究・保存
- **禁呪**: オーディン自身が封印した「知られざる言葉」

### 8.2 言語としての特徴

- **ガルドル**: 魔法詠唱のこと。声に出すことで初めて力を持つ
- **セイズル**: 女性的な魔法。男性が使うと社会的に蔑まれることも
- **ルーン刻印**: 文字を刻むことで静的な魔法効果を付与
- **ケニング**: 比喩表現。上級者は複雑なケニングで強力な魔法を構築
- **理解の深さ**: 意味を理解せず唱えても発動しない（言霊の本質）

### 8.3 発音ガイド
