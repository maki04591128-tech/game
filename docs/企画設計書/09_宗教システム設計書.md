## 1. システム概要

### 1.1 コンセプト

宗教は単なるバフ・スキル提供システムではなく、プレイスタイルを大きく変化させる要素。
信仰する神によって得られる恩恵だけでなく、禁忌（してはいけないこと）が発生し、
プレイヤーの行動に制約と戦略性を与える。

```
┌─────────────────────────────────────────────────────────────┐
│                    宗教システム概念図                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   【入信】                                                   │
│   宗教施設を訪問 → 入信儀式 → 信者となる                     │
│                                                             │
│   【信仰度】                                                 │
│   0 ──────────────────────────────── 100                    │
│   │     │        │        │        │                       │
│   無信仰  信者     敬虔     司祭     聖人                     │
│                                                             │
│   【恩恵と禁忌】                                             │
│   信仰度↑ → 恩恵増加 + 禁忌増加                              │
│                                                             │
│   【改宗】                                                   │
│   可能だが、前の宗教からの呪いを受ける                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 基本ルール

| 項目 | 内容 |
|------|------|
| 入信可能数 | 基本1つ（例外あり） |
| 信仰度範囲 | 0~100 |
| 改宗 | 可能（ペナルティあり） |
| 無神論 | 選択可能（特殊ルート） |
| 死に戻り | 信仰度80%で引き継ぎ |

### 3.1 光の神殿（Temple of Light）

#### 基本情報
| 項目 | 内容 |

| 本拠地 | 王都領・大聖堂 |
| 信者傾向 | 聖職者、騎士、貴族、一般市民 |

#### 入信条件
- 大聖堂で司祭に話しかける
- 闇の教団・死神信仰・混沌の崇拝に入信していないこと
- アンデッド・悪魔種族は入信不可

#### 禁忌一覧

|------|------------------|------|

| 闇魔法を使用する | -10 | 闇属性魔法 |
| 墓を荒らす | -10 | 墓地でのアイテム回収 |
| 嘘をつく | -5 | 特定会話選択肢 |
| 盗みを働く | -5 | 窃盗行為 |

#### 信仰度別恩恵
| 信仰度 | 階級 | 恩恵 |
|--------|------|------|

| 61~80 | 司祭 | 「聖光」スキル習得、信者を従者にできる |

| 100 | 聖人 | 「神の奇跡」発動可能、正気度回復+5 |

#### 専用スキル
| スキル | 必要信仰度 | 効果 |
|--------|------------|------|
| 聖光 | 60 | 周囲の闇を払い、アンデッドにダメージ |

### 3.6 無神論（Atheism）

#### 基本情報
| 項目 | 内容 |
|------|------|
| 主神 | なし |
| 教義 | 神に頼らず、己の力で運命を切り開く |
| 象徴 | なし |
| 本拠地 | なし（どこでも宣言可能） |
| 信者傾向 | 合理主義者、反逆者、自信家 |

#### 無神論選択条件
- ゲーム開始時、または任意のタイミングで宣言
- 既存の信仰がある場合は改宗扱い

#### 制約
| 制約 | 内容 |
|------|------|
| 宗教施設利用不可 | 回復、バフサービスを受けられない |
| 宗教NPC敵対 | 全ての宗教から軽蔑される |
| 神聖魔法使用不可 | 光・聖属性の一部魔法が使えない |

#### 無神論の恩恵
| 条件 | 恩恵 |
|------|------|
| 常時 | 全ステータス+5%、経験値+10% |
| 神殿を破壊時 | 強力なバフ獲得（一時的） |
| 信仰者を倒す | 追加経験値 |
| 神の化身を倒す | 専用装備ドロップ |
| 真エンディング | 無神論専用エンディング解放 |

#### 専用要素
- 「人の道」スキルツリー解放（信仰なしでも強化できるスキル群）
- 神殺しクエストライン
- 理性による正気度回復システム

## 4. 信仰度システム

### 4.1 信仰度の増減

#### 信仰度を上げる方法
| 行動 | 上昇量 | 備考 |
|------|--------|------|
| 宗教施設での祈り | +1~3/日 | 1日1回 |
| 宗教クエスト達成 | +5~20 | クエストによる |
| 敵対宗教の信者を倒す | +1~5 | 敵対関係による |
| 教義に沿った行動 | +1~3 | 各宗教で異なる |
| 布教活動 | +2~5 | NPCを改宗させる |
| 聖地巡礼 | +10 | 特定イベント |

#### 信仰度が下がる方法
| 行動 | 減少量 | 備考 |
|------|--------|------|
| 禁忌違反 | -5~30 | 上記参照 |
| 長期間祈らない | -1/10日 | |
| 敵対宗教への協力 | -10~20 | |
| 改宗 | 0にリセット | |

### 4.2 改宗システム

| 項目 | 内容 |
|------|------|
| 改宗可能 | いつでも可能 |
| 前宗教の信仰度 | 0にリセット |
| 改宗ペナルティ | 前宗教から「背教者の呪い」 |
| 再入信 | 可能だが、信仰度上限-20 |

#### 背教者の呪い
| 前宗教 | 呪いの内容 | 持続 |
