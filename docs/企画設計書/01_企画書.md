## 2. コンセプト

### 2.1 ゲームの概要
世界各地に現れたダンジョンを攻略し財宝や報酬を得て目的を達成する
構造が変化するダンジョンを死に覚えのトライアンドエラーで攻略する

### 2.2 コアコンセプト
様々な種族、クラス(職業)
周回回数によって減る正気度と使える手段の増加のバランスを考えどのように進めるかを考える戦略性
高難易度によるストレスを組み合わせのシナジーやプレイヤーの技量で突破するカタルシス

### 2.3 キャッチコピー
がんばれ!

## 3. ゲームプレイ

### 3.1 基本ルール
作成したキャラクターの素性による目標の達成がゲームのクリア条件
1ターンごとにマスの移動、アイテムの使用、攻撃、防御、スキルの使用等の豊富な選択肢を選び最適解を探す


### 3.2 操作方法
| 入力 | アクション |
|------|------------|
| W/A/S/D / 矢印キー | 移動（同時押しで斜め8方向移動） |
| Space | 待機（1ターン消費） |
| G | 足元のアイテムを拾う |
| I | インベントリ画面を開く |
| C | ステータス画面を開く |
| L | メッセージログ画面を開く |
| Tab | 自動探索（BFS経路探索、敵発見/HP低下等で自動停止） |
| M | ミニマップ表示/非表示 |
| Shift+. | 階段を降りる |
| Shift+, | 階段を上る |
| F | 周囲探索（隠しドア・罠発見） |
| X | ドアを閉じる |
| R | 遠隔攻撃（射撃） |
| T | アイテム投擲 |
| V | 魔法詠唱画面を開く |
| P | 祈り（宗教行動） |
| E | スキル使用 |
| J | ワールドマップ画面を開く |
| K | クエストログ画面を開く |
| O | 宗教画面を開く |
| H | 合成・鍛冶画面を開く |
| B | 街に入る |
| N | ギルド登録 |
| F5 | セーブ |
| F9 | ロード |
| Esc | ダイアログを閉じる |
| Q | ゲーム終了 |

### 3.3 ゲームサイクル
ターンごとに時間が経過し一定時間をオーバーするとゲームオーバーとなり死に戻りが発生する。
HPの全損や条件によるゲームオーバーになった場合、最初の場面に死に戻りする。
死に戻りが発生すると正気度が減少し、様々なバッドステータスが発生する。
死に戻りでは肉体の状態（ステータス）や持っていた装備などのアイテムは初期値に戻るが、取得したスキルや魔法、入った宗教、出会ったNPCの素性など知識は引き継がれる。

### 3.4 勝利条件・敗北条件
- **勝利条件**: 作成したキャラクターの素性による目的の達成
- **敗北条件**: HP全損、もしくはゲームオーバーイベント

## 4. 世界観・ストーリー

### 4.1 世界観設定
オーソドックスなダークファンタジー世界。舞台はヴァルディア王国という一つの国の中で完結し、6つの領地から構成される。
50年前に各地に突如出現した「裂け目」（ダンジョン）を攻略する冒険者の物語。

> **詳細**: `docs/18_設定資料.md` を参照

#### マップ階層詳細
| 階層 | 名称 | サイズ | 移動方式 | 時間経過 |
|------|------|--------|----------|----------|
| 第1階層 | 国（全体マップ） | 領地数分 | 領地選択 | 移動時に大きく経過 |
| 第2階層 | 領地（フィールドマップ） | ロケーション数分 | シンボル移動 | 移動時に経過 |
| 第3階層 | プレイマップ | 80×50マス（拡張予定） | ターン制グリッド | 1行動=1ターン |

#### 領地間移動
- 隣接する領地へのみ移動可能（一部例外あり）
- 移動時に大量のターンを消費（数千〜数万ターン）
- 移動中にランダムイベント発生の可能性

### 4.3 ストーリー概要
50年前に世界各地に突如出現した「裂け目」。凶悪な魔物が溢れ出す一方で、古代文明の財宝や失われた知識も発見された。
プレイヤーは「死に戻り」の能力を持つ冒険者として、死んでも知識を蓄積しながらダンジョン攻略に挑む。
キャラクター作成時に選ぶ「素性」によってクリア条件が変化し、放浪者は最深部到達、学者は古代文明の真実解明など多様な目的がある。

> **詳細**: `docs/18_設定資料.md` を参照

### 4.4 主要キャラクター
| キャラクター名 | 役割 | 説明 |
|----------------|------|------|
| プレイヤー（素性選択式） | 主人公 | 種族・クラス・背景を選択して作成。素性により目的が変化 |
| 死に戻りの案内人 | ガイドNPC | 死に戻りシステムを説明し、正気度の状態を教える存在 |
| 宗教の司祭たち | 宗教NPC | 各宗教の代表者。信仰することで特殊能力を得られる |
| ダンジョンの主 | ボス | 各ダンジョンの最深部に存在する強敵 |
| 放浪の商人 | 商人NPC | ダンジョン内外で出会う。周回で関係性が引き継がれる |
| 魔法言語の賢者 | 師匠NPC | 魔法言語の語彙を教えてくれる存在 |

## 6. 差別化ポイント

### 6.1 類似ゲームとの比較
| ゲーム名 | 共通点 | 差別化ポイント |
|----------|--------|----------------|
|不思議のダンジョン|構造が変わるダンジョン|ダンジョン以外の要素を追加|
|stone shard|ゲーム性|より豊富な選択肢、死=終了ではなくする|
|path of Accra|組み合わせシステム|より幅広い選択肢|

### 6.2 本作の独自性
ゲームオーバーは終わりではなく
魔法のシステム:魔法言語を学習し語彙の組み合わせによって発動内容を決定
ターン制限:1年(31,536,000ターン)以内に目的を達成する必要がある。制限の背景は素性ごとに異なり（ダンジョンのスタンピード、戦争、病気の流行等）、特定フラグの達成により延長・撤廃が可能（達成難度は周回前提）

## 8. リスクと対策

| リスク | 影響度 | 対策 |
|--------|--------|------|
| 1人開発による進捗遅延 | 高 | 早期アクセスで段階的リリース、MVPを明確化 |
| バランス調整の複雑化（組み合わせ爆発） | 高 | 早期からコミュニティテスト、調整用パラメータの外部化 |
| 正気度システムがストレスになりすぎる | 中 | 難易度選択の実装、救済措置の用意 |
| 魔法言語システムの学習コストが高い | 中 | チュートリアル充実、よく使う組み合わせのプリセット |
| アスキーアートの視認性問題 | 中 | カラーリング工夫、設定でフォント・色変更可能に |
| Steam審査・ローカライズ対応 | 低 | 英語を主言語として開発、日本語は後から追加 |

## 付録

### 参考資料
- 不思議のダンジョンシリーズ（ローグライクの基本構造）
- Stoneshard（ハードコアRPG、ターン制戦闘）
- Path of Achra（ビルド構築の多様性）
- Caves of Qud（魔法言語システムの参考）
- Darkest Dungeon（正気度・ストレスシステム）
- Elona（自由度の高いローグライク）
- DCSS (Dungeon Crawl Stone Soup)（種族・クラス・宗教システム）

### 備考
- 魔法言語仕様は別途「魔法言語設計書」として詳細化が必要
- 種族・クラス一覧は「キャラクター設計書」として分離予定
- 宗教システムは各宗教ごとの特典・制約を詳細設計する必要あり

### 追加検討事項
- **種族案**: 人間、エルフ、ドワーフ、オーク、獣人、ハーフリング、アンデッド、悪魔、堕天使、スライム（全10種族）
- **クラス案**: 戦士、騎士、盗賊、狩人、魔術師、僧侶、修道士、吟遊詩人、錬金術師、死霊術師
- **宗教案**: 光の神殿、闇の教団、自然崇拝、死神信仰、混沌の崇拝、無神論等
- **正気度低下時のバッドステータス案**:
  - 軽度: 幻覚（存在しない敵が見える）、被害妄想（NPC友好度低下）
  - 中度: 記憶混濁（マップ情報の一部消失）、暴走（一定確率で行動不能）
  - 重度: 自傷行為、NPC攻撃衝動、最終的に廃人化（実質ゲームオーバー）
