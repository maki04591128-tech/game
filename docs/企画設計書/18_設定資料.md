## 1. 世界概要

### 1.1 舞台

| 項目 | 内容 |
|------|------|
| 世界観 | オーソドックスなダークファンタジー |
| 舞台規模 | 一つの王国の中で完結 |
| 時代 | 冒険歴1024年（ゲーム内暦） |
| テーマ | 死と再生、狂気と理性、知識の蓄積 |
| トーン | 暗めだがユーモアもある、理不尽な死と成長のループ |

### 1.2 世界の特徴

- 50年前に各地に「裂け目」（ダンジョン）が出現
- ダンジョンからは魔物が溢れ、同時に財宝や古代の知識も発見される
- 冒険者ギルドが公認され、ダンジョン攻略が一大産業化
- 古代魔法文明の遺産を巡る争いが水面下で進行
- 「死に戻り」は世界の法則として存在する（プレイヤーだけの特権ではない可能性）

## 7. ストーリー概要

### 7.1 背景

50年前に世界各地に突然現れた「裂け目」——ダンジョン。
そこからは凶悪な魔物が溢れ出す一方で、古代文明の財宝や失われた知識も発見された。
王国はダンジョン攻略を合法化し、冒険者ギルドを公認。
多くの者がダンジョンに挑むが、その大半は帰らない。

### 7.2 プレイヤーの立場

プレイヤーは「死に戻り」の能力を持つ冒険者。
死んでも時間が巻き戻り、肉体は初期化されるが、知識と経験は蓄積される。
この特異な能力を活かし、繰り返しダンジョンに挑む。

### 7.3 素性とクリア条件

キャラクター作成時に選ぶ「素性」によって、ゲームの目的が変わる。

| 素性 | 実装名 | 目的 | 難易度 |
|------|--------|------|--------|
| 放浪者 | Wanderer | 最深部到達 | ★☆☆☆☆ |
| 復讐者 | Soldier | 仇敵の討伐 | ★★☆☆☆ |
| 学者 | Scholar | 古代文明の真実解明 | ★★★☆☆ |
| 亡命者 | Noble | 安住の地を見つける | ★★☆☆☆ |
| 聖職者 | Priest | 裂け目の封印 | ★★★★☆ |
| 商人 | Merchant | 莫大な財を築く | ★★☆☆☆ |
| 密偵 | Criminal | 王国の陰謀を暴く | ★★★★☆ |
| 罪人 | Penitent | 罪の贖い（特殊条件） | ★★★★★ |
| 傭兵 | Peasant | 名声を得る | ★★☆☆☆ |
| 預言者 | Adventurer | 世界の終末を阻止する | ★★★★★ |

> **詳細**: 素性のゲームプレイへの影響は `docs/05_キャラクター設計書.md` を参照

### 7.4 ターン制限と素性別の世界的危機

プレイヤーには **1年（31,536,000ターン）** の時間制限が課せられる。
これは単なるゲームメカニクスではなく、選んだ素性ごとに世界で進行する「取り返しのつかない危機」として物語に組み込まれている。
制限時間内にクリア条件を満たせなかった場合、世界は破滅を迎えゲームオーバーとなる。

#### 素性別のターン制限背景

| 素性 | 危機の内容 | 背景説明 |
|------|-----------|----------|
| 放浪者 | ダンジョンのスタンピード | ダンジョンの魔力が臨界に達し、1年後に大量の魔物が地上へ溢れ出す。最深部の核を破壊しなければ世界は魔物に蹂躙される |
| 復讐者 | 仇敵の逃亡計画 | 仇敵が1年後に国外へ逃亡する計画を進めている。時間切れで永遠に手が届かなくなる |
| 学者 | 古代遺跡の崩壊 | 古代文明の遺跡が風化・崩壊を続けており、1年後には真実を解き明かす手がかりが永久に失われる |
| 亡命者 | 追手の包囲網 | 故国の追手が1年以内に包囲網を完成させる。安住の地を確保しなければ捕縛・処刑される |
| 聖職者 | 裂け目の拡大 | 異界への裂け目が徐々に拡大しており、1年後に修復不能な規模に達する。封印の儀式を完遂しなければ世界が異界に呑まれる |
| 商人 | 経済崩壊の進行 | 戦争と疫病による経済崩壊が進行中。1年以内に莫大な財を築いて商業ギルドの再建資金としなければ、全領地の経済が破綻する |
| 密偵 | 王国転覆の陰謀 | 1年後にクーデターが決行される。それまでに陰謀の全容を暴き、阻止しなければ王国が滅ぶ |
| 罪人 | 処刑までの猶予 | 重罪人として1年の執行猶予を与えられている。贖いの条件を満たせなければ処刑が執行される |
| 傭兵 | 大戦争の勃発 | 領地間の緊張が高まり、1年後に全面戦争が勃発する。それまでに名声を確立し、戦争を調停できる立場に立たねばならない |
| 預言者 | 終末の預言成就 | 預言に記された世界の終末まで残り1年。全ての兆候を阻止し、終末の連鎖を断ち切らなければならない |

#### 素性別のターン制限延長・撤廃フラグ

特定の条件を達成することで、ターン制限を延長または撤廃できる。
ただし、**いずれの達成条件も周回（死に戻り）を前提とした難易度** に設定されている。
初回プレイで達成することは極めて困難であり、知識の蓄積と複数周回の経験が必要。

| 素性 | 延長フラグ | 延長効果 | 撤廃フラグ | 撤廃条件の難易度 |
|------|-----------|----------|-----------|----------------|
| 放浪者 | ダンジョン中層ボス撃破（3体中2体） | +半年（+15,768,000ターン） | 全階層ボス撃破＋核の封印 | ★★★★☆ |
| 復讐者 | 仇敵の協力者を3人排除 | +半年 | 仇敵のアジト特定＋逃亡ルート封鎖 | ★★★☆☆ |
| 学者 | 古代碑文の解読（6箇所中4箇所） | +半年 | 全碑文解読＋中枢遺跡への到達 | ★★★★☆ |
| 亡命者 | 3つの領地で協力者を確保 | +半年 | 全領地の安全拠点確立＋追手の指揮官排除 | ★★★☆☆ |
| 聖職者 | 裂け目の一部封印（6箇所中3箇所） | +半年 | 全裂け目封印＋異界の門番撃破 | ★★★★★ |
| 商人 | 3つの領地で交易路を開通 | +半年 | 全領地の経済圏統合＋商業ギルド再建完了 | ★★★☆☆ |
| 密偵 | 陰謀の協力者を3人特定 | +半年 | 全陰謀網の解明＋黒幕への直接接触 | ★★★★☆ |
| 罪人 | 贖いの試練を3つ達成 | +半年 | 全試練達成＋裁判官への弁明成功 | ★★★★★ |
| 傭兵 | 3つの領地で領主の信頼獲得 | +半年 | 全領地間の和平交渉成立＋調停者の称号獲得 | ★★★☆☆ |
| 預言者 | 終末の兆候を3つ阻止 | +半年 | 全兆候阻止＋終末の根源への到達 | ★★★★★ |

> **設計思想**: ターン制限は「緊張感の演出」と「周回の動機付け」を両立する仕組み。
> 延長フラグは中盤の目標として機能し、撤廃フラグは周回プレイの最終目標として位置づけられる。
> 死に戻りで知識が蓄積されることで、周回を重ねるほど効率的にフラグを達成できるようになる。

### 7.5 核心テーマ

- **死と再生のループ**: 死ぬたびに強くなるが、正気を失っていく
- **知識の蓄積**: 肉体は失われるが、知識だけが積み上がる
- **選択と代償**: 宗教、魔法、種族選択には全てトレードオフがある
- **世界の真実**: ダンジョンの正体、死に戻りの原因は最深部に隠されている

## 9. 魔法言語（ルーン語）の世界設定

- 古代魔法文明が使用していた言語体系
- 200年前の大魔法戦争で禁忌化されたが、ダンジョン内に碑文や書物が残る
- 単語の組み合わせで魔法効果が決定される（プログラミング的な魔法体系）
- 冒険者は死に戻りを通じてルーン語を少しずつ習得していく

> **詳細**: ルーン語の語彙・文法・効果については `docs/08_魔法言語設計書.md` を参照

## 11. 用語集

| 裂け目 | 50年前に出現したダンジョンの総称 |

| 冒険者ギルド | 王国公認の冒険者管理組織 |
| 素性 | キャラクター作成時に選ぶ背景設定。クリア条件を決定 |
