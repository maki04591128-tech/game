## 1. 経済・通貨・交易

### 1.1 通貨体系

| 通貨 | 実装名 | 価値 | 由来 | 流通範囲 |
|------|--------|------|------|---------|
| ヴァル金貨 | `Gold` | 基本通貨（1G） | 建国王ヴァルディウスの横顔を刻印 | 王国全土 |
| ヴァル銀貨 | — | 1/10G相当 | 簡易取引用、庶民の日常通貨 | 王国全土 |
| ヴァル銅貨 | — | 1/100G相当 | 最小単位、パンや水1個分 | 地方・村落 |
| ダンジョン原石 | — | 変動相場 | ダンジョンから産出される魔力結晶 | 冒険者間・闇市 |

> **設計上の簡略化**: ゲーム内では全てGold（ヴァル金貨）に換算して処理する。
> 銀貨・銅貨は世界観上の存在として、NPC台詞やフレーバーテキストに登場する。

### 1.2 通貨の歴史

- **建国時（約500年前）**: 建国王ヴァルディウスが統一通貨「ヴァル金貨」を制定。それ以前は各部族が独自の物々交換や宝石通貨を使用していた
- **大魔法戦争後（約200年前）**: 戦争による経済混乱で一時的に物々交換に逆戻り。戦後、賢者院が通貨鋳造の監督権を得て復興
- **裂け目出現後（50年前）**: ダンジョンから大量の財宝が流出し、一時的なインフレが発生。冒険者ギルドが財宝の買取価格を統制することで安定化

### 1.3 領地間の経済特性

| 領地 | 主要産業 | 特産品 | 物価傾向 | 交易上の特徴 |
|------|---------|--------|---------|-------------|
| 王都領 | 行政・商業 | 高級品全般 | 標準（基準） | 全領地の交易ハブ。何でも手に入るが割高 |
| 森林領 | 林業・薬草 | 世界樹の葉、エルフの弓 | やや高 | エルフ製品は高品質だが量が少ない |
| 山岳領 | 鉱業・鍛冶 | ミスリル鉱石、ドワーフの武具 | 武器安/食料高 | 武具は安いが食料は輸入に依存 |
| 沿岸領 | 漁業・造船 | 魚介、塩、珊瑚 | 食料安/武器高 | 海産物が豊富、内陸との交易で利益 |
| 南部領 | 農業・畜産 | 穀物、果実、ワイン | 食料安 | 王国の穀倉地帯、食料輸出の要 |
| 辺境領 | 傭兵・探索 | ダンジョン素材、魔物素材 | 高 | 物資は乏しいが貴重な素材が手に入る |
| 砂漠領 | 遊牧・交易 | 香辛料、異国の宝石、古代遺物 | 水高/装飾品安 | 水は非常に高価、南方交易路の起点 |
| 沼地領 | 薬学・呪術 | 毒薬・解毒剤、呪術道具 | 薬安/他高 | 特殊な素材の独占供給地 |
| 凍土領 | 狩猟・毛皮 | 毛皮、氷結晶、干し肉 | 高 | 厳しい環境ゆえに物資は常に不足 |
| 湖畔領 | 漁業・工芸 | 淡水真珠、精霊水、工芸品 | やや高 | 精霊との共生で得られる特殊素材 |
| 火山領 | 鍛冶・精錬 | 灼熱鉄、溶岩石、耐火素材 | 武器安/食料高 | 最高品質の鍛冶素材の産地 |
| 聖域 | 宗教・封印 | 聖水、聖遺物、祝福の品 | 非常に高 | 一般流通なし、信仰による奉納交換が中心 |

### 1.4 冒険者経済

| 要素 | 内容 |
|------|------|
| ダンジョン素材の買取 | 冒険者ギルドが公定価格で買取。市場価格の70〜90% |
| 税制度 | ダンジョン素材の売却益に10%の「裂け目税」を賦課。領地の防衛費に充当 |
| 闇市場 | カルマが低い（Rogue以下）場合にのみ利用可能。禁制品・盗品の売買 |
| 保険制度 | 冒険者ギルドが任意の装備保険を提供（死に戻り時に装備の一部を回収可能） |
| 報奨金制度 | クエスト達成時の報奨金はギルドランクに応じて増額される |

### 1.5 闇経済

- **密輸ルート**: 沿岸領→砂漠領の海路が主要な密輸経路。禁制魔法道具や非合法薬物が流通
- **闇ギルド**: 犯罪者素性のプレイヤーが関わることができる裏組織。盗品の換金率が高い
- **禁制品**: 大魔法戦争時代の兵器魔法、アンデッド召喚触媒、正気度に作用する薬物

### 2.3 政治的緊張関係

```
              王家（アルドリック3世）
             ╱        │         ╲
        支持派       中立        反発派
     ┌────┴────┐     │     ┌────┴────┐
   山岳領    火山領  沿岸領  凍土領   砂漠領
                     南部領
                     湖畔領

  独立志向: 森林領（エルフ長老会）
  孤立  : 沼地領（モルガ）
  特殊  : 聖域（宗教的独立）、辺境領（実力主義で王家への忠誠薄い）
```

#### 主要な政治的対立

1. **王位継承問題**: 王太子セドリック vs 王弟ガレス公爵。密偵素性のメインストーリーに直結
2. **裂け目対策の路線対立**: 軍事的封鎖（セドリック派）vs ギルドに委任（現状維持派）vs 共存（学者派）
3. **辺境領の独立機運**: 「鉄拳」グラムは王家からの独立をほのめかす。最も多くのダンジョンを抱え、辺境領なしでは王国の防衛が成り立たない
4. **エルフ長老会の不信**: 大魔法戦争時に森林領が最も被害を受けた歴史があり、人間の魔法使用を今も警戒している
5. **凍土領の不満**: 最も環境が厳しいにもかかわらず、王都からの支援が最小限。ウルリッヒは独自の交易路を模索

### 2.4 冒険者ギルドと王権の関係

| 側面 | 内容 |
|------|------|
| 法的位置づけ | 王国の勅令により公認された独立組織。王権からの直接命令は受けない |
| 実質的関係 | 相互依存。王国は裂け目対策をギルドに依存し、ギルドは王国の法的保護を必要とする |
| 緊張要素 | ギルドの独立性が高まりすぎると、王権を脅かす第二の権力になる懸念がある |
| ギルドマスター | ヴァレン（元A級冒険者）。政治には関わらない方針だが、王太子から軍事協力を迫られている |

### 2.5 宗教と政治の関係

| 宗教 | 政治的立場 | 詳細 |
|------|-----------|------|
| 光の神殿 | 王家支持 | 王家の正統性を宗教的に裏付ける。国王の戴冠式は大司教が執り行う。王太子セドリックの軍事路線を支持する急進派も存在 |
| 闇の教団 | 反体制 | 表向きは非政治的だが、深淵の使徒との関係が疑われる。南部領の裏社会に浸透し、王弟ガレスの一派と接触の噂あり |
| 自然崇拝 | エルフ長老会と連動 | 森林領・湖畔領のエルフ社会と一体化。人間の政治には関心が薄いが、森林に影響する政策には激しく反対 |
| 死神信仰 | 中立（秘密結社的） | 表立った政治参加はないが、山岳領のドワーフ社会に根を張る。死に戻り現象について独自の見解を持つ |
| 混沌の崇拝 | 反王家 | 辺境領の混沌の祭壇を拠点に活動。王家の秩序を否定し、「強さこそ正義」を説く。「鉄拳」グラムとの関係が深い |
| 無神論 | 個人の信念 | 組織的な政治勢力ではないが、賢者院の研究者に多い。宗教的権威の政治介入に批判的 |

> **参照**: 各宗教の教義・組織の詳細は `09_宗教システム設計書.md` を参照。

## 4. 死に戻りの世界的認知と影響

### 4.1 一般的な認識

| 層 | 認識 | 詳細 |
|----|------|------|
| 一般市民 | ほぼ無知 | 死に戻りの存在を知らない。「冒険者は運が良い」程度の認識 |
| 冒険者ギルド | 噂レベル | 「何度死んでも戻ってくる冒険者がいるらしい」という噂。公式には認めていない |
| 賢者院 | 研究対象 | 死に戻り現象を「時間退行現象」として極秘裏に研究中。プレイヤーを被験者として観察している可能性 |
| 宗教界 | 教義により異なる | 光の神殿は「神の恩寵」、死神信仰は「死の循環の一部」、闇の教団は「禁忌の力」と解釈 |
| 王家 | 把握している | 少数の「死に戻り者」の存在を知っているが、政治的に利用価値があるため黙認 |

### 4.2 死に戻りの法則（世界設定上の仮説）

| 仮説 | 提唱者 | 内容 | ストーリー上の位置づけ |
|------|--------|------|---------------------|
| 神の恩寵説 | 光の神殿 | 太陽神ソラリスが選ばれし者に与える祝福 | 聖職者素性のメインストーリーで掘り下げ |
| 世界の修復機構説 | 賢者院 | 裂け目による世界のほころびを修復するため、世界自体が死に戻りを生み出した | 学者素性のメインストーリーで掘り下げ |
| 古代文明の遺産説 | 学者NPC（アルベルト） | 古代魔法文明が作った「時間の輪」の残滓 | 碑文の解読で示唆される |
| 深淵の王の罠説 | 辺境の傭兵（ウルフ） | 深淵の王が冒険者を誘い込むための罠。死ぬたびに正気を吸い取られている | エンドコンテンツで明かされる真実の一部 |

### 4.3 他の死に戻り者

- **レオン（案内人NPC）**: プレイヤーに死に戻りを「授ける」と述べている。彼自身も死に戻りの経験者である可能性が示唆される
- **過去の英雄**: 歴史上の英雄の中には「何度も死地から生還した」と記録される人物がおり、彼らも死に戻り者だった可能性がある
- **深淵の王**: エンドコンテンツで明かされるが、深淵の王自体が「死に戻りの無限ループに囚われた存在」である可能性

### 4.4 正気度低下の世界設定的意味

| 正気度段階 | 実装名 | 世界設定上の解釈 |
|-----------|--------|----------------|
| 正常（80-100） | `Normal` | 時間退行の負荷を自然に処理できる状態 |
| 不安（60-79） | `Uneasy` | 繰り返しの記憶が蓄積し、微細な違和感を覚え始める |
| 焦燥（40-59） | `Anxious` | 「この世界は本当に現実なのか」という疑念が生じる |
| 不安定（20-39） | `Unstable` | 過去の記憶と現在の区別がつかなくなる。幻覚・幻聴 |
| 狂気（1-19） | `Madness` | 世界の裏側——「深淵」が見え始める。異常な力を得るが制御不能 |
| 廃人化（0） | `Broken` | 自我の完全な崩壊。肉体は動くが魂は深淵に呑まれている |

> **設計思想**: 正気度の低下は「繰り返すことの代償」を体現する。
> プレイヤーは知識を蓄積するが、その代償として現実感を失っていく。
> これは「ゲームを周回プレイする」というメタ的な行為と重なるよう設計されている。

### 5.3 ダンジョンの生態系

- **魔物の出自**: 裂け目から出現する魔物は、元々は深淵界の住人。裂け目を通じて地上に適応した個体が定着
- **素材の循環**: ダンジョン内では通常の物理法則が変容するため、地上では存在し得ない鉱石や植物が生成される
- **ダンジョンの「成長」**: 裂け目は放置すると徐々に拡大する。冒険者がダンジョンの魔物を間引き、核を抑制することで拡大を防いでいる
- **領地ごとの裂け目の特徴**: 裂け目は出現した土地の性質に影響される（例：火山領の裂け目は炎属性の魔物が多い）

### 8.3 大魔法戦争の真実

公式の歴史:
> 約200年前、各領地の魔法使いが軍事目的でルーン語を悪用。
> 制御不能な破壊魔法が各地で暴走し、多くの犠牲を出した。
> 戦後、賢者院が設立されルーン語は厳しく規制された。

隠された真実（ゲーム中に断片的に明かされる）:
> 大魔法戦争は偶発的な暴走ではなく、「深淵界」からの侵食に対抗するための
> 最終手段だった。古代文明の技術を使って深淵界への裂け目を一時的に封じたが、
> その代償として大地が荒廃した。賢者院はこの真実を隠蔽し、
> 「人間の愚かさが引き起こした戦争」という物語を作り上げた。

## 10. エンドコンテンツの世界観的位置づけ

### 10.1 深淵の王の正体

| 段階 | 明かされる情報 |
|------|-------------|
| 序盤 | 「深淵の王」という名前のみ。辺境の最大ダンジョンの最深部にいるとされる |
| 中盤 | 古代文明が封印した存在。裂け目の原因とされる |
| 後半 | 実は古代魔法文明の最高指導者だった。世界を「完璧に」するため、時間操作の禁呪を使い、世界ごと深淵に引きずり込もうとした |
| 最深部 | 深淵の王は「死に戻りの無限ループに囚われた存在」。何千年もの間、自身の過ちを繰り返し続けている。プレイヤーの死に戻りと同じ仕組み |
| 真実 | プレイヤーの死に戻り能力は、深淵の王の力の一部が漏れ出したもの。死ぬたびに正気度が減るのは、深淵の王に少しずつ「呑まれている」から |

### 10.2 最終選択（エンディング分岐）

メインストーリーの最終局面で、プレイヤーは以下の選択を迫られる：

| 選択 | 対応する素性 | 結末 | 条件 |
|------|------------|------|------|
| 剣で断つ | 放浪者・復讐者・傭兵 | 深淵の王を倒し、裂け目を閉じる。だが死に戻りの力も失われる | 戦闘力が最高水準 |
| 知識で封じる | 学者・預言者 | 古代のルーン語を使い、深淵の王を再封印する。裂け目は残るが安定する | 全碑文の解読、ルーン語マスタリー |
| 信仰で浄化する | 聖職者 | 信仰の力で深淵の王を浄化し、元の人間に戻す。裂け目は消滅する | 最高信仰段階（聖人） |
| 闇と共に在る | 犯罪者（密偵） | 深淵の王と契約し、その力を受け継ぐ。プレイヤーが新たな「深淵の王」となる | カルマ最低＋深淵の使徒と協力 |
| 全てを受け入れる | 罪人 | 自らの正気度を代償に深淵の王を吸収。世界は救われるが、プレイヤーは廃人化する | 正気度を自発的に0にする |
| 無に帰す | — | 深淵の王もプレイヤーも、裂け目ごと消滅させる。世界は平和になるが、ダンジョン攻略という文化も失われる | 全素性の全条件達成（真エンディング） |

### 10.3 全ボス撃破後の世界変化

| 変化 | 内容 |
|------|------|
| 裂け目の安定化 | 領地ボスを倒すことで、その領地の裂け目が「安定」し、魔物の地上出現率が低下 |
| NPC反応の変化 | 「英雄」として扱われる。物価低下、クエスト報酬増加 |
| 新エリアの解放 | 深淵の裂け目の45F以降が開放される可能性（隠しボス・隠しエンディング） |
| 世界の変容 | 全領地ボス撃破後、世界全体の色調が明るくなる演出。NPCの会話も「希望」を語るようになる |

### 10.4 無限ダンジョンの設定的意味

| 項目 | 内容 |
|------|------|
| 名称 | 「終わりなき裂け目」（The Endless Rift） |
| 設定上の位置づけ | メインダンジョン攻略後に出現する、深淵界そのものへの入口 |
| 世界観上の説明 | 深淵の王を倒した（または封じた）ことで、深淵界への直接通路が開いた |
| 構造 | 階層は無限に続き、深くなるほど深淵界の法則が強くなる |
| 意味 | 「冒険に終わりはない」というゲームのテーマを体現。死に戻りの力が残る限り、冒険者は挑み続ける |

### 10.5 真エンディングの示す「世界の真実」

> 真エンディングに到達した場合にのみ明かされる設定：
>
> - この世界自体が「深淵の王が作り出した閉じた時間の輪」の中にある
> - 裂け目は「外の世界」への通路であり、深淵界こそが本来の世界
> - 死に戻りは世界の修復機構ではなく、「この輪を維持するための仕組み」
> - プレイヤーが全ての条件を達成し「無に帰す」を選ぶと、時間の輪が破壊され、世界は「本来の時間」に戻る
> - つまり、プレイヤーは「ゲームの周回構造そのもの」を終わらせる
>
> **メタ的な意味**: 真エンディングは「ローグライクの周回プレイ」というジャンルの構造自体を
> ストーリーに組み込んだものであり、プレイヤーが「もうこのゲームを繰り返す必要はない」と
> 気づく瞬間を描いている。
