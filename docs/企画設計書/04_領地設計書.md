## 1. 国の概要

### 1.1 基本情報

| 項目 | 内容 |
|------|------|
| 国名 | ヴァルディア王国（Kingdom of Valdia） |
| 統治形態 | 封建君主制（各領地に領主が存在） |
| 首都 | 王都クラウンヘイム（Crownheim） |
| 領地数 | 12領地 |
| 気候 | 温帯〜寒帯（北部ほど寒冷） |

### 1.2 歴史概要

| 時代 | 出来事 |
|------|--------|
| 古代 | 先住民族と魔法文明の繁栄、大崩壊により滅亡 |
| 500年前 | 建国王ヴァルディウスが諸部族を統一、王国建国 |
| 200年前 | 大魔法戦争、魔法言語の禁忌化と賢者院の設立 |
| 100年前 | 北部の反乱と鎮圧、辺境領の成立 |
| 50年前 | 「裂け目」の出現、ダンジョンが各地に発生 |
| 現在 | ダンジョンからの財宝を求める冒険者時代 |

### 1.3 現在の情勢

- 各地に出現した「裂け目（ダンジョン）」から魔物が溢れ、同時に財宝も発見される
- 冒険者ギルドが公認され、ダンジョン攻略が産業化
- 領主間の権力闘争が水面下で進行中
- 古代魔法文明の遺産を巡る争い

## 4. ロケーション種別定義

### 4.1 種別一覧

| 種別 | 説明 | プレイマップ生成 |
|------|------|------------------|
| 街 | 大規模な居住地、多様な施設 | 固定マップ |
| 村 | 小規模な居住地、限定的な施設 | 固定マップ |
| 施設 | 特定機能を持つ建物・場所 | 固定マップ |
| 宗教施設 | 宗教関連の場所 | 固定マップ |
| フィールド | 野外エリア、探索・戦闘 | 半固定+ランダム要素 |
| ダンジョン | 危険な探索エリア | 自動生成（シード固定可） |

### 4.2 プレイマップ生成ルール

> **📌 Ver.prt.0.12 変更点**
> - 町・村・フィールドマップからStairsUp（上り階段）を撤去し、EntrancePosition方式に統一
> - 施設・宗教施設マップはPlaceExitDoor経由のStairsUpを維持（Shift+<で退出）
> - 各領の非ダンジョンロケーションを統合街エントリに集約（SubLocationIds）

#### 固定マップ
- 街・村・施設は固定レイアウト
- NPC配置、店の位置は固定
- イベントフラグで一部変化

#### 半固定マップ
- フィールドは地形の大枠は固定
- 敵配置、アイテム配置はランダム
- 天候・時間帯で変化

#### 自動生成マップ
- ダンジョンは入る度に構造が変化
- シード値を保存することで同じ構造を再現可能
- 階層が深いほど難易度上昇
- 特定フロアにボス部屋（固定構造）

## 5. 時間・環境システム

### 5.1 時間帯

| 時間帯 | ターン範囲（1日=86,400ターン） | 影響 |
|--------|-------------------------------|------|
| 早朝 | 0 - 14,400 | 店開店準備、野生動物活発 |
| 昼 | 14,400 - 43,200 | 通常状態、最も安全 |
| 夕方 | 43,200 - 57,600 | 店閉店準備、一部敵出現 |
| 夜 | 57,600 - 72,000 | 店閉店、夜行性の敵出現 |
| 深夜 | 72,000 - 86,400 | 最も危険、アンデッド活性化 |

### 5.2 天候

| 天候 | 発生確率 | 影響 |
|------|----------|------|
| 晴れ | 50% | 通常状態 |
| 曇り | 25% | 視界やや低下 |
| 雨 | 15% | 視界低下、火属性弱体化、移動コスト増 |
| 嵐 | 5% | 視界大幅低下、野外活動危険 |
| 霧 | 5% | 視界最低、奇襲されやすい |

### 5.3 領地ごとの特殊環境

| 領地 | 特殊環境 | 効果 |
|------|----------|------|
| 沿岸領 | 嵐（梅雨期） | 視界低下、移動コスト増 |
| 凍土領 | 吹雪（冬季） | 寒冷ダメージ、凍結リスク |
| 南部領 | 熱波（夏季） | 疲労蓄積増加 |
| 森林領 | 瘴気（腐敗の森） | 毒・病気リスク |
| 山岳領 | 高山病（高地） | スタミナ消費増加 |
| 砂漠領 | 砂嵐（季節問わず） | 視界激減、砂ダメージ |
| 沼地領 | 瘴気（常時） | 毒・病気リスク |
| 火山領 | 噴火（ランダム） | 火属性ダメージ、移動制限 |
| 聖域 | 聖なる光（常時） | アンデッド弱体化、正気度回復 |
