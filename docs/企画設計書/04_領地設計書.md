## 1. 国の概要

### 1.1 基本情報

| 項目 | 内容 |
|------|------|
| 国名 | ヴァルディア王国（Kingdom of Valdia） |
| 統治形態 | 封建君主制（各領地に領主が存在） |
| 首都 | 王都クラウンヘイム（Crownheim） |
| 領地数 | 6領地 |
| 気候 | 温帯〜寒帯（北部ほど寒冷） |

### 1.2 歴史概要

| 時代 | 出来事 |
|------|--------|
| 古代 | 先住民族と魔法文明の繁栄、大崩壊により滅亡 |
| 500年前 | 建国王ヴァルディウスが諸部族を統一、王国建国 |
| 200年前 | 大魔法戦争、魔法言語の禁忌化と賢者院の設立 |
| 100年前 | 北部の反乱と鎮圧、辺境領の成立 |
| 50年前 | 「裂け目」の出現、ダンジョンが各地に発生 |
| 現在 | ダンジョンからの財宝を求める冒険者時代 |

### 1.3 現在の情勢

- 各地に出現した「裂け目（ダンジョン）」から魔物が溢れ、同時に財宝も発見される
- 冒険者ギルドが公認され、ダンジョン攻略が産業化
- 領主間の権力闘争が水面下で進行中
- 古代魔法文明の遺産を巡る争い

## 4. ロケーション種別定義

### 4.1 種別一覧

| 種別 | 説明 | プレイマップ生成 |
|------|------|------------------|
| 街 | 大規模な居住地、多様な施設 | 固定マップ |
| 村 | 小規模な居住地、限定的な施設 | 固定マップ |
| 施設 | 特定機能を持つ建物・場所 | 固定マップ |
| 宗教施設 | 宗教関連の場所 | 固定マップ |
| フィールド | 野外エリア、探索・戦闘 | 半固定+ランダム要素 |
| ダンジョン | 危険な探索エリア | 自動生成（シード固定可） |

### 4.2 プレイマップ生成ルール

#### 固定マップ
- 街・村・施設は固定レイアウト
- NPC配置、店の位置は固定
- イベントフラグで一部変化

#### 半固定マップ
- フィールドは地形の大枠は固定
- 敵配置、アイテム配置はランダム
- 天候・時間帯で変化

#### 自動生成マップ
- ダンジョンは入る度に構造が変化
- シード値を保存することで同じ構造を再現可能
- 階層が深いほど難易度上昇
- 特定フロアにボス部屋（固定構造）

## 5. 時間・環境システム

### 5.1 時間帯

| 時間帯 | ターン範囲（1日=86,400ターン） | 影響 |
|--------|-------------------------------|------|
| 早朝 | 0 - 14,400 | 店開店準備、野生動物活発 |

```
      │    【同盟】
      │          │
      ↓          ↓
    王都領 ←───→ 南部領
   （中立）    【友好】
      │
      │【緊張】
      ↓
    北部領 ←──→ 辺境領
         【協力】
    （対魔物共同戦線）
```

### 6.2 関係性がゲームに与える影響

| 関係 | 影響 |
|------|------|
| 友好 | 通行自由、商品値引き、クエスト発生 |
| 中立 | 通常通り |
| 緊張 | 検問発生、一部施設利用不可 |
| 敵対 | 通行に条件、発覚時戦闘、密入国イベント |
